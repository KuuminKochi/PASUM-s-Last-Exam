Tutorial 3: Buffers and Titrations

Overview

This tutorial covers buffer solutions, buffer calculations, salt hydrolysis, qualitative analysis of salt solutions, and titration curves with pH calculations.

Topics: Buffer solutions, Henderson-Hasselbalch equation, salt hydrolysis, titration curves, indicators Difficulty: Medium Estimated Time: 150 minutes

---

Question 1 [10 marks]

State the definition of buffer solution.

Show solution

Buffer Solution: A solution that maintains its pH when small amounts of acid or base are added to it.

A buffer consists of a weak acid and its conjugate base (or weak base and its conjugate acid).

Examples: Acetic acid + sodium acetate, ammonia + ammonium chloride

Question 2 [25 marks]

A one liter of a buffer solution, which was prepared by dissolving 0.0250 mol of formic acid and 0.018 mol of sodium formate has a pH value of 3.63. If 5.0 mL of 0.10 M NaOH is added, what is the new pH value of this buffer solution? By comparing both pH values, what can you conclude? (Ka for formic acid = 1.7 √ó 10‚Åª‚Å¥)

Show solution

Given:

Formic acid (HCOOH): 0.0250 mol

Sodium formate (HCOONa): 0.018 mol

Initial pH = 3.63

Ka = 1.7 √ó 10‚Åª‚Å¥

Added NaOH: 5.0 mL √ó 0.10 M = 0.0005 mol

Step 1: Verify initial pH using Henderson-Hasselbalch

$$pK_a = -\log(1.7 \times 10^{-4}) = 3.77$$

$$pH = pK_a + \log\left(\frac{[A^-]}{[HA]}\right)$$

$$3.63 = 3.77 + \log\left(\frac{0.018}{0.0250}\right)$$

$$\log\left(\frac{0.018}{0.0250}\right) = -0.14$$

$$\frac{0.018}{[0.0250]} = 0.724$$

This is consistent with the given pH.

Step 2: Calculate change after NaOH addition

NaOH reacts with formic acid:

$$HCOOH + OH^- \rightarrow HCOO^- + H_2O$$

Moles of HCOOH after reaction = 0.0250 - 0.0005 = 0.0245 mol

Moles of HCOO‚Åª after reaction = 0.018 + 0.0005 = 0.0185 mol

Step 3: Calculate new pH

$$pH = pK_a + \log\left(\frac{[A^-]}{[HA]}\right)$$

$$pH = 3.77 + \log\left(\frac{0.0185}{0.0245}\right)$$

$$pH = 3.77 + \log(0.755)$$

$$pH = 3.77 + (-0.122)$$

$$pH = \boxed{3.65}$$

Conclusion: The pH change is very small (from 3.63 to 3.65) - only 0.02 units. This demonstrates the buffer effect: buffers resist changes in pH when small amounts of acid or base are added.

Critical Nuance: If we had added the same amount of NaOH to pure water, the pH would have changed dramatically. The buffer maintains pH by consuming the added OH‚Åª through reaction with the weak acid.

Question 4 [40 marks]

Sketch a titration curve and suggest a suitable indicator for the titration between 25.0 mL of 0.50 M NaOH with 0.50 M HF. Calculate the pH of:

Given: Ka for hydrofluoric acid = 7.1 √ó 10‚Åª‚Å¥

Part (a) [5 marks]

the initial NaOH

Show solution

Initial NaOH is a strong base with no acid present.

Since there's no acid added yet, the pH is determined solely by the NaOH concentration.

However, we need to consider that NaOH is a strong base:

$$[OH^-] = 0.50 \text{ M}$$

$$pOH = -\log(0.50) = 0.30$$

$$pH = 14 - 0.30 = \boxed{13.70}$$

Note: This is the pH of 0.50 M NaOH solution before any acid is added.

Part (b) [6 marks]

when 15 mL of HF is added

Show solution

Step 1: Calculate moles

Moles of NaOH = 0.0250 L √ó 0.50 = 0.0125 mol

Moles of HF added = 0.015 L √ó 0.50 = 0.0075 mol

Step 2: Determine reaction

NaOH + HF ‚Üí NaF + H‚ÇÇO

NaOH is in excess by: 0.0125 - 0.0075 = 0.0050 mol

Step 3: Calculate [OH‚Åª]

Total volume = 25.0 + 15.0 = 40.0 mL = 0.040 L

[OH‚Åª] = 0.0050 / 0.040 = 0.125 M

Step 4: Calculate pH

$$pOH = -\log(0.125) = 0.903$$

$$pH = 14 - 0.903 = \boxed{13.10}$$

Part (c) [6 marks]

when 24.5 mL of HF is added

Show solution

Step 1: Calculate moles

Moles of NaOH = 0.0125 mol (unchanged)

Moles of HF = 0.0245 L √ó 0.50 = 0.01225 mol

Step 2: Excess

Excess NaOH = 0.0125 - 0.01225 = 0.00025 mol

Step 3: [OH‚Åª]

Total volume = 25.0 + 24.5 = 49.5 mL = 0.0495 L

[OH‚Åª] = 0.00025 / 0.0495 = 5.05 √ó 10‚Åª¬≥ M

Step 4: pH

$$pOH = -\log(5.05 \times 10^{-3}) = 2.30$$

$$pH = 14 - 2.30 = \boxed{11.70}$$

Part (d) [8 marks]

at equivalence point

Show solution

Step 1: Find volume at equivalence point

Moles of NaOH = 0.0125 mol

Volume of HF needed = 0.0125 / 0.50 = 0.0250 L = 25.0 mL

Step 2: At equivalence point

All NaOH reacts with HF to form NaF:

$$NaOH + HF \rightarrow NaF + H_2O$$

Moles of NaF formed = 0.0125 mol

Total volume = 25.0 + 25.0 = 50.0 mL = 0.0500 L

[NaF] = 0.0125 / 0.0500 = 0.25 M

Step 3: Hydrolysis of F‚Åª

F‚Åª is the conjugate base of HF:

$$K_b = \frac{K_w}{K_a} = \frac{1.0 \times 10^{-14}}{7.1 \times 10^{-4}} = 1.41 \times 10^{-11}$$

$$F^- + H_2O \rightleftharpoons HF + OH^-$$

Let x = [OH‚Åª]

$$K_b = \frac{x^2}{0.25 - x} = 1.41 \times 10^{-11}$$

Since Kb is very small, x << 0.25

$$x^2 = 1.41 \times 10^{-11} \times 0.25 = 3.53 \times 10^{-12}$$

$$x = 1.88 \times 10^{-6} \text{ M}$$

Step 4: pH

$$pOH = -\log(1.88 \times 10^{-6}) = 5.73$$

$$pH = 14 - 5.73 = \boxed{8.27}$$

Critical Nuance: The equivalence point is slightly basic because F‚Åª (conjugate base of weak acid) hydrolyzes to produce OH‚Åª.

Part (e) [7 marks]

when 25.5 mL of HF is added

Show solution

Step 1: Calculate moles

Moles of NaOH = 0.0125 mol

Moles of HF added = 0.0255 L √ó 0.50 = 0.01275 mol

Step 2: Excess HF

Excess HF = 0.01275 - 0.0125 = 0.00025 mol

Step 3: Volume

Total volume = 25.0 + 25.5 = 50.5 mL = 0.0505 L

[HF] = 0.00025 / 0.0505 = 4.95 √ó 10‚Åª¬≥ M

Step 4: Calculate pH using weak acid formula

For HF: Ka = 7.1 √ó 10‚Åª‚Å¥

Let x = [H‚Å∫]

$$K_a = \frac{x^2}{[HF] - x} = \frac{x^2}{4.95 \times 10^{-3}}$$

$$x^2 = 7.1 \times 10^{-4} \times 4.95 \times 10^{-3} = 3.51 \times 10^{-6}$$

$$x = 1.87 \times 10^{-3} \text{ M}$$

$$pH = -\log(1.87 \times 10^{-3}) = \boxed{2.73}$$

Part (f) [8 marks]

when 50 mL of HF is added

Show solution

Step 1: Calculate moles

Moles of NaOH = 0.0125 mol

Moles of HF added = 0.050 L √ó 0.50 = 0.025 mol

Step 2: Excess HF

Excess HF = 0.025 - 0.0125 = 0.0125 mol

Step 3: Volume

Total volume = 25.0 + 50.0 = 75.0 mL = 0.0750 L

[HF] = 0.0125 / 0.0750 = 0.167 M

Step 4: pH

$$K_a = \frac{x^2}{0.167 - x}$$

$$x^2 = 7.1 \times 10^{-4} \times 0.167 = 1.19 \times 10^{-4}$$

$$x = 1.09 \times 10^{-2} \text{ M}$$

$$pH = -\log(1.09 \times 10^{-2}) = \boxed{1.96}$$

Suggested Indicator:

The equivalence point pH is 8.27 (slightly basic). Suitable indicators would be:

  • Phenolphthalein (pH range 8.2-10.0) - most suitable
  • Thymol blue (pH range 8.0-9.6)

Since the equivalence point is in the basic region, phenolphthalein is the best choice.
