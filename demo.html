Question 1 60 marks

Find the indefinite integrals for the following functions.

Part (a) 5 marks
$$\\int (3e^{4x} - 5e^{-6x}) \\, dx$$
üéØ Exponential integration, linearity üìê Key: $\\int e^{ax} \\, dx = \\frac{1}{a}e^{ax} + C$
‚ñ∂ Need a hint?
üí° Hint
Split the integral using linearity: $\\int (f(x) - g(x)) \\, dx = \\int f(x) \\, dx - \\int g(x) \\, dx$.
Remember that $\\int e^{ax} \\, dx = \\frac{1}{a}e^{ax} + C$. Watch out for the negative coefficient!
Method: Direct integration using exponential rule
$$\\begin{aligned} \\int (3e^{4x} - 5e^{-6x}) \\, dx &= \\frac{3e^{4x}}{4} - \\frac{5e^{-6x}}{-6} + C \\\\ &= \\boxed{\\frac{3}{4}e^{4x} + \\frac{5}{6}e^{-6x} + C} \\end{aligned}$$
‚ö†Ô∏è Critical Nuance
Watch the sign when integrating $e^{-6x}$. The negative exponent becomes positive in the denominator, but the negative sign in front of 5 makes it positive overall. Students often miss this and get $-\\frac{5}{6}$ instead of $+\\frac{5}{6}$.

Part (b) 5 marks
$$\\int (6x^6 + 4x^3 + 8) \\, dx$$
üéØ Power rule for integration
‚ñ∂ Need a hint?
üí° Hint
Use the power rule: $\\int x^n \\, dx = \\frac{x^{n+1}}{n+1} + C$ for each term.
Remember to add 1 to the exponent, then divide by the new exponent.
Method: Power rule for integration
$$\\begin{aligned} \\int (6x^6 + 4x^3 + 8) \\, dx &= \\frac{6x^7}{7} + \\frac{4x^4}{4} + 8x + C \\\\ &= \\boxed{\\frac{6}{7}x^7 + x^4 + 8x + C} \\end{aligned}$$
‚ö†Ô∏è Critical Nuance
When integrating $x^3$, the exponent becomes 4, not $4x^4$. Divide by the new exponent.
Part (f) 5 marks
$$\\int \\frac{\\sin^2 x + \\cos^2 x}{3x} \\, dx$$
üéØ Trigonometric identity, logarithmic integration
‚ñ∂ Need a hint?
üí° Hint
Before integrating, look at the numerator. Is there a trigonometric identity that simplifies $\\sin^2 x + \\cos^2 x$?

Once simplified, you'll have $\\frac{1}{3x}$ which is a standard logarithmic integral.
Method: Trigonometric identity + logarithmic integration
$$\\begin{aligned} \\int \\frac{\\sin^2 x + \\cos^2 x}{3x} \\, dx &= \\int \\frac{1}{3x} \\, dx \\quad \\text{(since } \\sin^2 x + \\cos^2 x = 1\\text{)} \\\\ &= \\frac{1}{3} \\int \\frac{1}{x} \\, dx \\\\ &= \\boxed{\\frac{1}{3}\\ln|x| + C} \\end{aligned}$$
‚ö†Ô∏è Critical Nuance
Many students miss the identity $\\sin^2 x + \\cos^2 x = 1$. Without this, the problem becomes much harder. Always look for simplifications before integrating!
Question 2 40 marks

Use appropriate substitution to find:

Part (a) 4 marks
$$\\int 3x^2(x^3 - 2)^3 \\, dx$$
üéØ u-substitution
‚ñ∂ Need a hint?
üí° Hint
Try $u = x^3 - 2$. Calculate $du$ and see if $3x^2 \\, dx$ appears in the integral.

Remember: $du = \\frac{du}{dx} \\, dx = 3x^2 \\, dx$
Method: u-substitution
$$\\begin{aligned} &\\text{Let } u = x^3 - 2, \\quad du = 3x^2 \\, dx, \\quad x^2 \\, dx = \\frac{du}{3} \\\\ &\\int 3x^2(x^3 - 2)^3 \\, dx = \\int 3 \\cdot u^3 \\cdot \\frac{du}{3} = \\int u^3 \\, du \\\\ &= \\frac{u^4}{4} + C = \\boxed{\\frac{(x^3 - 2)^4}{4} + C} \\end{aligned}$$
‚ö†Ô∏è Critical Nuance
Notice that $3x^2$ is the derivative of $x^3 - 2$. This makes substitution straightforward. Look for this pattern: $f(g(x)) \\cdot g'(x)$
Part (j) 4 marks
$$\\int \\frac{x^3 + 2x^2 + 5x + 6}{x + 2} \\, dx$$
üéØ Polynomial long division, then integration
‚ñ∂ Need a hint?
üí° Hint
When the degree of the numerator is greater than or equal to the degree of the denominator, perform polynomial long division first.

Divide $x^3 + 2x^2 + 5x + 6$ by $x + 2$.
Method: Polynomial long division
$$\\begin{aligned} &\\frac{x^3 + 2x^2 + 5x + 6}{x + 2} = (x^2 + 5) - \\frac{4}{x + 2} \\\\ &\\int \\frac{x^3 + 2x^2 + 5x + 6}{x + 2} \\, dx = \\int \\left(x^2 + 5 - \\frac{4}{x + 2}\\right) \\, dx \\\\ &= \\frac{x^3}{3} + 5x - 4\\ln|x + 2| + C \\end{aligned}$$
‚ö†Ô∏è Critical Nuance
When degree of numerator $\\geq$ degree of denominator, perform polynomial long division first! This is a common mistake - students try to use partial fractions or other methods when division is the correct first step.

Frequency: High | Seen in: Q1a, Q1j

Root Cause: Confusion when integrating $e^{-ax}$
Strategy: Write out $\\frac{1}{-a} = -\\frac{1}{a}$ explicitly

Frequency: High | Seen in: Q1f, Q1j

Root Cause: Trying to integrate before simplifying
Strategy: Always look for cancellations and identities first

Strategy: Look for $f(g(x)) \\cdot g'(x)$ structure
