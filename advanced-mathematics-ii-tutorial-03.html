Tutorial 3: Complex Logarithms & Loci

Question 1 [6 marks]

Find general complex logarithm $\ln(z)$ and principal complex logarithm $\Ln(z)$ for: (a) $z = 1 + i$ (b) $z = 2 - \sqrt{3}i$ (c) $z = 2\sqrt{3} + 2i$

Need a hint?

$\ln(z) = \ln|z| + i(\theta + 2k\pi)$ where $\theta = \arg(z)$. Principal: $\Ln(z) = \ln|z| + i\arg(z)$ with $\arg \in (-\pi, \pi]$.

Show solution (a) $z = 1 + i$: $|z| = \sqrt{2}$, $\arg(z) = \pi/4$

$\ln(z) = \ln\sqrt{2} + i(\pi/4 + 2k\pi)$, $k \in \mathbb{Z}$

$\Ln(z) = \ln\sqrt{2} + i\pi/4$

(b) $z = 2 - \sqrt{3}i$: $|z| = \sqrt{4 + 3} = \sqrt{7}$, $\arg(z) = -\pi/3$

$\ln(z) = \ln\sqrt{7} + i(-\pi/3 + 2k\pi)$

$\Ln(z) = \ln\sqrt{7} - i\pi/3$

(c) $z = 2\sqrt{3} + 2i$: $|z| = \sqrt{12 + 4} = 4$, $\arg(z) = \pi/6$

$\ln(z) = \ln 4 + i(\pi/6 + 2k\pi)$

$\Ln(z) = \ln 4 + i\pi/6$

‚ö†Ô∏è Nuance: The complex logarithm is multi-valued due to the $2k\pi$ term. The principal value uses the principal argument.

Question 2 [8 marks]

(a) Find Cartesian form of $\sin z$ and $\cos z$. (b) Compute: (i) $\cos(\pi/4 + i)$ (ii) $\sin(\pi/4 + i)$

Need a hint?

Use definitions: $\sin z = \frac{e^{iz} - e^{-iz}}{2i}$, $\cos z = \frac{e^{iz} + e^{-iz}}{2}$. Then use $\sin(A \pm iB) = \sin A \cosh B \pm i\cos A \sinh B$.

Show solution (a) Let $z = x + iy$:

$\sin z = \sin(x+iy) = \sin x \cosh y + i \cos x \sinh y$

$\cos z = \cos(x+iy) = \cos x \cosh y - i \sin x \sinh y$

(b)(i) $\cos(\pi/4 + i) = \cos(\pi/4)\cosh(1) - i\sin(\pi/4)\sinh(1)$

$= \frac{\sqrt{2}}{2}\cosh(1) - i\frac{\sqrt{2}}{2}\sinh(1)$

(b)(ii) $\sin(\pi/4 + i) = \sin(\pi/4)\cosh(1) + i\cos(\pi/4)\sinh(1)$

$= \frac{\sqrt{2}}{2}\cosh(1) + i\frac{\sqrt{2}}{2}\sinh(1)$

Using compound angle formula gives same result.

‚ö†Ô∏è Nuance: Unlike real trig, complex sine and cosine can be > 1 in magnitude. Here $\cosh(1) \approx 1.54$ and $\sinh(1) \approx 1.17$.

Question 3 [3 marks]

Find $\text{Re}(z)$, $\text{Im}(z)$ and $\arg(z)$ for $z = 2e^{i\pi/3}$.

Need a hint?

Convert to Cartesian: $z = 2(\cos\pi/3 + i\sin\pi/3) = 2(1/2 + i\sqrt{3}/2) = 1 + i\sqrt{3}$.

Show solution

$z = 2e^{i\pi/3} = 2(\cos\pi/3 + i\sin\pi/3) = 1 + i\sqrt{3}$

$\text{Re}(z) = 1$

$\text{Im}(z) = \sqrt{3}$

$\arg(z) = \pi/3$

‚ö†Ô∏è Nuance: Euler's form $re^{i\theta}$ directly gives modulus $r$ and argument $\theta$.

Question 4 [6 marks]

Sketch and find Cartesian equation for: (a) $|z - (4 + i)| = 3$ (b) $|z - 1 - i| = 5$ (c) $|2z + 6 - 4i| = 6$

Need a hint?

These are circles. $|z - z_0| = r$ means circle centered at $z_0$ with radius $r$.

Show solution (a) $|z - (4 + i)| = 3$

Let $z = x + iy$: $|(x+iy) - (4+i)| = 3$

$|(x-4) + i(y-1)| = 3$

$\sqrt{(x-4)^2 + (y-1)^2} = 3$

$(x-4)^2 + (y-1)^2 = 9$

Circle: center (4, 1), radius 3

(b) $|z - 1 - i| = 5$

$(x-1)^2 + (y-1)^2 = 25$

Circle: center (1, 1), radius 5

(c) $|2z + 6 - 4i| = 6$

$|2(z + 3 - 2i)| = 6$ ‚Üí $|z + 3 - 2i| = 3$

$(x+3)^2 + (y-2)^2 = 9$

Circle: center (-3, 2), radius 3

‚ö†Ô∏è Nuance: For (c), factor out the coefficient of z first.

Question 5 [6 marks]

Find Cartesian equation for perpendicular bisectors: (a) $|z - (2 + i)| = |z - (1 + 3i)|$ (b) $|z + 2 - i| = |z - 1 + 3i|$ (c) $|z + 1 + i| = 1$

Need a hint?

Perpendicular bisector: set of points equidistant from two points. Simplify to form $Ax + By + C = 0$.

Show solution (a) $|z - (2 + i)| = |z - (1 + 3i)|$

$|(x-2) + i(y-1)| = |(x-1) + i(y-3)|$

$(x-2)^2 + (y-1)^2 = (x-1)^2 + (y-3)^2$

$x^2 - 4x + 4 + y^2 - 2y + 1 = x^2 - 2x + 1 + y^2 - 6y + 9$

$-4x - 2y + 5 = -2x - 6y + 10$

$-2x + 4y - 5 = 0$

$2x - 4y + 5 = 0$

(b) Similarly: gives $3x - 2y + 1 = 0$ (c) $|z + 1 + i| = 1$

$(x+1)^2 + (y+1)^2 = 1$

Circle: center (-1, -1), radius 1

‚ö†Ô∏è Nuance: The perpendicular bisector is a straight line.

Question 6 [6 marks]

Find Cartesian equation for half-lines: (a) $\arg(z - (2 - i)) = \pi/4$ (b) $\arg(z + 2 - i) = 5\pi/6$ (c) $\arg(z + 2 + i) = -\pi/3$

Need a hint?

$\arg(z - z_0) = \theta$ is a half-line (ray) from $z_0$ at angle $\theta$.

Show solution (a) $\arg(z - (2 - i)) = \pi/4$

Let $z - (2 - i) = re^{i\pi/4} = r(\cos\pi/4 + i\sin\pi/4) = r/\sqrt{2}(1 + i)$

So $x - 2 + i(y + 1) = r/\sqrt{2}(1 + i)$

Thus $x - 2 = y + 1$ ‚Üí $x - y = 3$

With $x > 2$ (to get the ray, not the opposite line)

(b) $\arg(z + 2 - i) = 5\pi/6$

$\tan(5\pi/6) = -1/\sqrt{3}$

Equation: $y + 1 = -\frac{1}{\sqrt{3}}(x + 2)$

With appropriate domain restriction

(c) $\arg(z + 2 + i) = -\pi/3$

$\tan(-\pi/3) = -\sqrt{3}$

$y + 1 = -\sqrt{3}(x + 2)$

With appropriate domain restriction

‚ö†Ô∏è Nuance: The half-line starts at the point, not at the origin.

Question 7 [4 marks]

Find equations for: (a) Circle radius 5, center $z = 3 - 2i$ (b) Perpendicular bisector of segment from $z = -1 - 2i$ to $z = 3 - 2i$ (c) Half-line at angle $\alpha$ from point $(-1, -2)$

Need a hint?

Use formulas from Q4-6.

Show solution (a) Circle: $|z - (3 - 2i)| = 5$

Cartesian: $(x-3)^2 + (y+2)^2 = 25$

(b) Points: $(-1, -2)$ and $(3, -2)$. Midpoint: $(1, -2)$. Line is vertical: $x = 1$

In complex: $|z - (1 - 2i)| = |z - (3 - 2i)|$

(c) Half-line: $\arg(z - (-1 - 2i)) = \alpha$, i.e., $\arg(z + 1 + 2i) = \alpha$

Cartesian: $\tan\alpha = \frac{y+2}{x+1}$, with $x > -1$

‚ö†Ô∏è Nuance: The perpendicular bisector of horizontal segment is a vertical line.

Common Patterns

Complex Logarithm:

$\ln(z) = \ln|z| + i(\arg z + 2k\pi)$, $k \in \mathbb{Z}$

Circle:

$|z - z_0| = r$ ‚Üí $(x-x_0)^2 + (y-y_0)^2 = r^2$

Perpendicular Bisector:

$|z - z_1| = |z - z_2|$ ‚Üí linear equation

Half-line (Ray):

$\arg(z - z_0) = \theta$
