Tutorial 3: Discrete Random Variables, PDF and CDF

Overview

This tutorial covers discrete random variables, probability distribution functions (PDF), and cumulative distribution functions (CDF).

Question 1 [9 marks]

Determine if the following are valid probability functions:

Part (a) [3 marks]

$$P(X = x) = \frac{2x-1}{16}, \quad x = 1, 2, 3, 4$$
Show solution

Check: (1) All probabilities ‚â• 0, (2) Sum = 1

$P(1) = \frac{1}{16}$, $P(2) = \frac{3}{16}$, $P(3) = \frac{5}{16}$, $P(4) = \frac{7}{16}$

Sum: $\frac{1+3+5+7}{16} = \frac{16}{16} = 1$ ‚úì

All probabilities positive ‚úì

Valid probability function

Part (b) [3 marks]

$$P(X = x) = \frac{x^3}{15}, \quad x = -1, 0, 1, 2$$
Show solution

$P(-1) = \frac{-1}{15} < 0$ ‚úó

Not a valid probability function (probabilities must be non-negative)

Part (c) [3 marks]

$$P(X = x) = \frac{2}{3(1+2x)}, \quad x = 0, 1, 2$$
Show solution

$P(0) = \frac{2}{3}$, $P(1) = \frac{2}{9}$, $P(2) = \frac{2}{15}$

Sum: $\frac{2}{3} + \frac{2}{9} + \frac{2}{15} = \frac{30+10+6}{45} = \frac{46}{45} > 1$ ‚úó

Not a valid probability function (sum ‚â† 1)

Question 2 [9 marks]

X has probability distribution:

$$P(X = x) = \begin{cases} 0.1 & \text{for } x = 2, 4, 6, 8 \\ 0.2 & \text{for } x = 1, 3, 5 \\ 0 & \text{otherwise} \end{cases}$$

Part (a) [3 marks]

Show X is a discrete random variable

Show solution

X takes countable values (1, 2, 3, 4, 5, 6, 8) with associated probabilities

Sum: $4(0.1) + 3(0.2) = 0.4 + 0.6 = 1$ ‚úì

X is a discrete random variable

Part (b) [2 marks]

Find $P(X = 4)$

Show solution

$P(X = 4) = \boxed{0.1}$

Part (c) [2 marks]

Find $P(X < 4)$

Show solution

$X < 4$ means $X = 1, 2, 3$

$P = 0.2 + 0.1 + 0.2 = \boxed{0.5}$

Part (d) [2 marks]

Find $P(3 \leq X < 6)$

Show solution

$3 \leq X < 6$ means $X = 3, 4, 5$

$P = 0.2 + 0.1 + 0.2 = \boxed{0.5}$

Question 3 [12 marks]

X has probability function:

$$P(X = x) = \begin{cases} 0.1 & \text{for } x = -1, 0, 5 \\ a & \text{for } x = 1, 3 \\ 0.3 & \text{for } x = 4 \\ 0 & \text{otherwise} \end{cases}$$

Part (a) [3 marks]

Write probability distribution table in terms of a

Show solution
x-101345
P(X=x)0.10.1aa0.30.1

Part (b) [3 marks]

Find a

Show solution

Sum of probabilities = 1:

$0.1 + 0.1 + a + a + 0.3 + 0.1 = 1$

$0.6 + 2a = 1$

$2a = 0.4$

$a = \boxed{0.2}$

Part (c) [3 marks]

Find $P(X \geq 3)$

Show solution

$X \geq 3$ means $X = 3, 4, 5$

$P = 0.2 + 0.3 + 0.1 = \boxed{0.6}$

Part (d) [3 marks]

Find the CDF F(x)

Show solution

Order the values: -1, 0, 1, 3, 4, 5

$$F(x) = \begin{cases} 0 & x < -1 \\ 0.1 & -1 \leq x < 0 \\ 0.2 & 0 \leq x < 1 \\ 0.4 & 1 \leq x < 3 \\ 0.6 & 3 \leq x < 4 \\ 0.9 & 4 \leq x < 5 \\ 1 & x \geq 5 \end{cases}$$

Question 4 [12 marks]

Drawer: 8 white socks, 4 yellow socks. Alisha takes 2 socks one after another.

Part (a) [4 marks]

Show P(one white and one yellow) = $\frac{16}{33}$

Show solution

Method 1: W then Y + Y then W

$= \frac{8}{12} \times \frac{4}{11} + \frac{4}{12} \times \frac{8}{11}$

$= \frac{32}{132} + \frac{32}{132} = \frac{64}{132} = \boxed{\frac{16}{33}}$ ‚úì

Part (b) [8 marks]

Y = number of white socks taken. Find PDF and CDF of Y.

Show solution

PDF:

$P(Y=0)$ (both yellow): $\frac{4}{12} \times \frac{3}{11} = \frac{12}{132} = \frac{1}{11}$

$P(Y=1)$ (one white, one yellow): $\frac{16}{33}$ (from part a)

$P(Y=2)$ (both white): $\frac{8}{12} \times \frac{7}{11} = \frac{56}{132} = \frac{14}{33}$

CDF:

$$F(y) = \begin{cases} 0 & y < 0 \\ \frac{1}{11} & 0 \leq y < 1 \\ \frac{1}{11} + \frac{16}{33} = \frac{19}{33} & 1 \leq y < 2 \\ 1 & y \geq 2 \end{cases}$$

Find k

Show solution

$0.2 + 0.25 + 0.4 + k + 0.05 = 1$

Question 7 [13 marks]

X has CDF:

$$F(x) = \begin{cases} 0 & x < 0.1 \\ 0.05 & 0.1 \leq x < 0.2 \\ 0.3 & 0.2 \leq x < 0.3 \\ 0.6 & 0.3 \leq x < 0.4 \\ 0.75 & 0.4 \leq x < 0.5 \\ 1 & x \geq 0.5 \end{cases}$$

Part (a) [6 marks]

Find:

(i) [2 marks]

$P(X \leq 0.4)$

Show solution

$P(X \leq 0.4) = F(0.4) = \boxed{0.75}$

(ii) [2 marks]

$P(X > 0.3)$

Show solution

$P(X > 0.3) = 1 - P(X \leq 0.3) = 1 - F(0.3) = 1 - 0.6 = \boxed{0.4}$

(iii) [2 marks]

$P(X = 0.3)$

Show solution

$P(X = 0.3) = F(0.3) - F(0.3^-) = 0.6 - 0.3 = \boxed{0.3}$

Common Patterns Identified

Pattern 1: Validating PDF ‚Äî High frequency (Q1)
Strategy: Check (1) all P(X=x) ‚â• 0, (2) sum of all probabilities = 1

Pattern 2: Finding Unknown Constant ‚Äî High frequency (Q3b, Q5a)
Strategy: Use fact that total probability = 1 to solve for unknown

Pattern 3: Converting PDF to CDF ‚Äî High frequency
Strategy: CDF F(x) = P(X ‚â§ x) = sum of all probabilities for values ‚â§ x

Pattern 4: Converting CDF to PDF ‚Äî Medium frequency (Q7b)
Strategy: P(X=x) = F(x) - F(x‚Åª) = jump in CDF at x
