Tutorial 2: De Moivre's Theorem

Question 1 [4 marks]

Let $z_1 = -4 + 4i$ and $z_2 = -2\sqrt{3} + 2i$. Write in Cartesian form: (a) $z_1^6$ (b) $z_2^6$

Need a hint?

First convert to polar form, then use De Moivre's theorem: $z^n = r^n(\cos n\theta + i\sin n\theta)$.

Show solution (a) $z_1 = -4 + 4i$

$r = \sqrt{(-4)^2 + 4^2} = \sqrt{32} = 4\sqrt{2}$

$\theta = \pi - \pi/4 = 3\pi/4$

$z_1^6 = (4\sqrt{2})^6 [\cos(6 \times 3\pi/4) + i\sin(6 \times 3\pi/4)]$

$= (4\sqrt{2})^6 [\cos(9\pi/2) + i\sin(9\pi/2)]$

$= 4096[\cos(\pi/2 + 4\pi) + i\sin(\pi/2 + 4\pi)]$

$= 4096[0 + i(1)] = 4096i$

(b) $z_2 = -2\sqrt{3} + 2i$

$r = \sqrt{(-2\sqrt{3})^2 + 2^2} = \sqrt{12 + 4} = 4$

$\theta = \pi - \pi/6 = 5\pi/6$

$z_2^6 = 4^6 [\cos(6 \times 5\pi/6) + i\sin(6 \times 5\pi/6)]$

$= 4096[\cos(5\pi) + i\sin(5\pi)]$

$= 4096[-1 + i(0)] = -4096$

‚ö†Ô∏è Nuance: When simplifying angles, remember to reduce by $2\pi$ to get the principal angle.

Question 2 [6 marks]

Simplify to terms in cos x and sin x: (a) $(\cos 7x + i\sin 7x)(\cos 5x - i\sin 5x)$ (b) $\frac{\cos 3x + i\sin 3x}{\cos 2x + i\sin 2x}$ (c) $(\cos 4x + i\sin 4x)(\cos 2x + i\sin 2x)(\cos x + i\sin x)$

Need a hint?

Use De Moivre's theorem. For (b), divide complex numbers in polar form. For (c), multiply the angles.

Show solution (a) $(\cos 7x + i\sin 7x)(\cos 5x - i\sin 5x) = (\cos 7x + i\sin 7x)(\cos(-5x) + i\sin(-5x))$

Using formula: $= \cos(7x - 5x) + i\sin(7x - 5x) = \cos 2x + i\sin 2x$

(b) $\frac{\cos 3x + i\sin 3x}{\cos 2x + i\sin 2x} = \cos(3x - 2x) + i\sin(3x - 2x) = \cos x + i\sin x$ (c) Product of three: $\cos(4x + 2x + x) + i\sin(4x + 2x + x) = \cos 7x + i\sin 7x$

‚ö†Ô∏è Nuance: Remember: $\cos(-\theta) = \cos\theta$ and $\sin(-\theta) = -\sin\theta$. Division corresponds to subtracting arguments.

Question 3 [4 marks]

Express in exponential and Cartesian form: (a) $(\cos\frac{\pi}{6} + i\sin\frac{\pi}{6})^4$ (b) $(\cos\frac{2\pi}{3} + i\sin\frac{2\pi}{3})^5$

Need a hint?

Use De Moivre's theorem: $(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$.

Show solution (a) $(\cos\frac{\pi}{6} + i\sin\frac{\pi}{6})^4 = \cos\frac{4\pi}{6} + i\sin\frac{4\pi}{6} = \cos\frac{2\pi}{3} + i\sin\frac{2\pi}{3}$

Exponential: $e^{i4\pi/6} = e^{i2\pi/3}$

Cartesian: $\cos(2\pi/3) + i\sin(2\pi/3) = -1/2 + i\sqrt{3}/2$

(b) $(\cos\frac{2\pi}{3} + i\sin\frac{2\pi}{3})^5 = \cos\frac{10\pi}{3} + i\sin\frac{10\pi}{3}$

Reduce: $10\pi/3 = 2\pi + 4\pi/3$

$= \cos\frac{4\pi}{3} + i\sin\frac{4\pi}{3}$

Exponential: $e^{i10\pi/3}$

Cartesian: $\cos(4\pi/3) + i\sin(4\pi/3) = -1/2 - i\sqrt{3}/2$

‚ö†Ô∏è Nuance: Simplify angles by subtracting multiples of $2\pi$.

Question 4 [4 marks]

Calculate: (a) $\arg\left[\frac{\cos\theta + i\sin\theta}{\cos\phi + i\sin\phi}\right]$ (b) modulus and argument of $\frac{\cos 2\theta + i\sin 2\theta}{\cos\theta + i\sin\theta}$

Need a hint?

Use the property: $\arg(z_1/z_2) = \arg(z_1) - \arg(z_2)$. For modulus: $|z_1/z_2| = |z_1|/|z_2|$.

Show solution (a) $\arg\left[\frac{\cos\theta + i\sin\theta}{\cos\phi + i\sin\phi}\right] = \arg(\cos\theta + i\sin\theta) - \arg(\cos\phi + i\sin\phi) = \theta - \phi$ (b) $\frac{\cos 2\theta + i\sin 2\theta}{\cos\theta + i\sin\theta} = \cos(2\theta - \theta) + i\sin(2\theta - \theta) = \cos\theta + i\sin\theta$

Modulus = 1, Argument = $\theta$ (or $\theta + 2k\pi$)

‚ö†Ô∏è Nuance: The modulus of any $\cos\theta + i\sin\theta$ is always 1, since $\cos^2\theta + \sin^2\theta = 1$.

Question 5 [6 marks]

Show that $\cos A + \cos A = 2\cos A$ and $\cos A - \cos A = 2i\sin A$. Deduce: (a) $\cos n\theta + \cos(-n\theta) = 2\cos(n\theta)$ (b) $\cos n\theta - \cos(-n\theta) = 2i\sin(n\theta)$

Hence express $\cos 4x$ and $\sin 4x$ in terms of powers of $\cos x$ and $\sin x$.

Need a hint?

Use sum-to-product identities. For the deduction, note that $\cos(-n\theta) = \cos(n\theta)$ and $\sin(-n\theta) = -\sin(n\theta)$.

Show solution

Using $e^{i\theta} = \cos\theta + i\sin\theta$:

$\cos n\theta + \cos(-n\theta) = \frac{e^{in\theta} + e^{-in\theta}}{2} + \frac{e^{in\theta} + e^{-in\theta}}{2} = e^{in\theta} + e^{-in\theta} = 2\cos(n\theta)$

$\cos n\theta - \cos(-n\theta) = \frac{e^{in\theta} + e^{-in\theta}}{2} - \frac{e^{in\theta} + e^{-in\theta}}{2} = 0$... wait, let me redo:

$\cos n\theta - \cos(-n\theta) = \frac{e^{in\theta} + e^{-in\theta}}{2} - \frac{e^{-in\theta} + e^{in\theta}}{2}$... actually:

Using $\cos\theta = \frac{e^{i\theta} + e^{-i\theta}}{2}$ and $\sin\theta = \frac{e^{i\theta} - e^{-i\theta}}{2i}$:

$\cos n\theta + \cos(-n\theta) = 2\cos(n\theta)$

$\cos n\theta - \cos(-n\theta) = 0$? No, this is wrong.

Actually: $\cos(-n\theta) = \cos(n\theta)$, so: $\cos n\theta + \cos(-n\theta) = \cos n\theta + \cos n\theta = 2\cos(n\theta)$

And: $\cos n\theta - \cos(-n\theta) = \cos n\theta - \cos n\theta = 0$? That's also wrong.

Wait - the question says: show $\cos A + \cos A = 2\cos A$ - that's trivial. The actual identity is:

$\cos A + \cos B = 2\cos(\frac{A+B}{2})\cos(\frac{A-B}{2})$

$\cos A - \cos B = -2\sin(\frac{A+B}{2})\sin(\frac{A-B}{2})$

For $\cos 4x$: Use double angle repeatedly or $e^{i4x}$.

$\cos 4x = \cos^4 x - 6\cos^2 x \sin^2 x + \sin^4 x$

$\sin 4x = 4\cos^3 x \sin x - 4\cos x \sin^3 x$

‚ö†Ô∏è Nuance: Using De Moivre's theorem: $(\cos x + i\sin x)^4 = \cos 4x + i\sin 4x$. Expand using binomial theorem.

Question 6 [6 marks]

Prove: (a) $\cos 5\theta = 16\cos^5\theta - 20\cos^3\theta + 5\cos\theta$ (b) $\sin 5\theta = 16\sin^5\theta - 20\sin^3\theta + 5\sin\theta$

Need a hint?

Use $(\cos\theta + i\sin\theta)^5 = \cos 5\theta + i\sin 5\theta$ and expand using binomial theorem.

Show solution

Using $(\cos\theta + i\sin\theta)^5$:

$= \cos^5\theta + 5i\cos^4\theta\sin\theta + 10i^2\cos^3\theta\sin^2\theta + 10i^3\cos^2\theta\sin^3\theta + 5i^4\cos\theta\sin^4\theta + i^5\sin^5\theta$

$= \cos^5\theta + 5i\cos^4\theta\sin\theta - 10\cos^3\theta\sin^2\theta - 10i\cos^2\theta\sin^3\theta + 5\cos\theta\sin^4\theta + i\sin^5\theta$

Real part (cos 5Œ∏): $\cos 5\theta = \cos^5\theta - 10\cos^3\theta\sin^2\theta + 5\cos\theta\sin^4\theta$

Using $\sin^2\theta = 1 - \cos^2\theta$: $= \cos^5\theta - 10\cos^3\theta(1-\cos^2\theta) + 5\cos\theta(1-\cos^2\theta)^2$

$= \cos^5\theta - 10\cos^3\theta + 10\cos^5\theta + 5\cos\theta - 10\cos^3\theta + 5\cos^5\theta$

$= 16\cos^5\theta - 20\cos^3\theta + 5\cos\theta$ ‚úì

Imaginary part gives (b): $\sin 5\theta = 5\cos^4\theta\sin\theta - 10\cos^2\theta\sin^3\theta + \sin^5\theta$

Using $\cos^2\theta = 1 - \sin^2\theta$: $= 5(1-\sin^2\theta)^2\sin\theta - 10(1-\sin^2\theta)\sin^3\theta + \sin^5\theta$

$= 16\sin^5\theta - 20\sin^3\theta + 5\sin\theta$ ‚úì

‚ö†Ô∏è Nuance: This pattern continues: $\cos n\theta$ can be expressed as a polynomial in $\cos\theta$ of degree n.

Question 7 [6 marks]

Find cube roots of: (a) $z = -3 + 3i$ (b) $z = 8(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3})$ (c) $z = 27$

Need a hint?

For $z^n = w$, write $w = re^{i\theta}$, then roots are $r^{1/n}e^{i(\theta + 2k\pi)/n}$ for $k = 0, 1, ..., n-1$.

Show solution (a) $-3 + 3i$: $r = \sqrt{9+9} = 3\sqrt{2}$, $\theta = 3\pi/4$

Cube roots ($n=3$, $k=0,1,2$): $r^{1/3} = (3\sqrt{2})^{1/3}$, angles: $(3\pi/4 + 2k\pi)/3$

$k=0$: $3^{1/3}2^{1/6}e^{i\pi/4} = \sqrt[3]{3\sqrt{2}}e^{i\pi/4}$ $k=1$: angle = $(3\pi/4 + 2\pi)/3 = 11\pi/12$ $k=2$: angle = $(3\pi/4 + 4\pi)/3 = 19\pi/12$

(b) $8(\cos\pi/3 + i\sin\pi/3)$: $r = 8$, $\theta = \pi/3$

Cube roots: $8^{1/3} = 2$, angles: $(\pi/3 + 2k\pi)/3$

$k=0$: $2e^{i\pi/9}$ $k=1$: $2e^{i7\pi/9}$ $k=2$: $2e^{i13\pi/9}$

(c) $27 = 27 + 0i$: $r = 27$, $\theta = 0$

Cube roots: $27^{1/3} = 3$, angles: $0, 2\pi/3, 4\pi/3$

$3$, $3e^{i2\pi/3} = -3/2 + i3\sqrt{3}/2$, $3e^{i4\pi/3} = -3/2 - i3\sqrt{3}/2$

‚ö†Ô∏è Nuance: There are always n roots for an nth root problem. They are equally spaced on the circle.

Question 8 [4 marks]

What are the 10th roots of 1? How many are on: real axis? imaginary axis? first quadrant? other quadrants?

Need a hint?

The 10th roots of 1 are $e^{i2k\pi/10} = e^{ik\pi/5}$ for $k = 0, 1, ..., 9$.

Show solution

10th roots of 1: $e^{ik\pi/5}$ for $k = 0, 1, ..., 9$

These are equally spaced points on the unit circle.

Real axis: Points where $\theta = 0$ or $\pi$: $k=0$ (1) and $k=5$ (-1) ‚Üí 2 points Imaginary axis: Points where $\theta = \pi/2$ or $3\pi/2$: none among 10th roots ‚Üí 0 points First quadrant: $\theta$ between 0 and $\pi/2$: $k=1$ ($\pi/5 = 36^\circ$) and $k=2$ ($2\pi/5 = 72^\circ$) ‚Üí 2 points Other quadrants: Remaining 6 points

‚ö†Ô∏è Nuance: The n roots of 1 are evenly spaced at angles $2k\pi/n$.

Question 9 [4 marks]

Find all $z$ for which $z^5 = -32i$ with $\text{Im}(z) > 0$.

Need a hint?

Write $-32i$ in polar form: $r = 32$, $\theta = -\pi/2$ (or $3\pi/2$). Then find 5th roots.

Show solution

$-32i = 32e^{-i\pi/2}$ (or $32e^{i3\pi/2}$)

5th roots: $32^{1/5} = 2$, angles: $(-\pi/2 + 2k\pi)/5$

For $k = 0, 1, 2, 3, 4$: - $k=0$: $2e^{-i\pi/10}$ ‚Üí $\text{Im} < 0$ - $k=1$: $2e^{i3\pi/10}$ ‚Üí $\text{Im} > 0$ ‚úì - $k=2$: $2e^{i7\pi/10}$ ‚Üí $\text{Im} > 0$ ‚úì - $k=3$: $2e^{i11\pi/10}$ ‚Üí $\text{Im} < 0$ - $k=4$: $2e^{i15\pi/10} = 2e^{i3\pi/2}$ ‚Üí $\text{Im} < 0$

Solutions with Im(z) > 0: $z = 2e^{i3\pi/10}$ and $z = 2e^{i7\pi/10}$

‚ö†Ô∏è Nuance: Exactly 2 of the 5 roots will have positive imaginary part.

Question 10 [4 marks]

Suppose $\sqrt{3} - i$ is a root of $z^4 + 16(1+i)z^2 + a + bi = 0$, where $a, b$ are real. Find $a$ and $b$.

Need a hint?

If $\sqrt{3} - i$ is a root, substitute into the polynomial. Since coefficients are real, the complex conjugate is also a root.

Show solution

Let $z = \sqrt{3} - i$. First find $z^2$ and $z^4$:

$z = \sqrt{3} - i = 2e^{-i\pi/6}$

$z^2 = 4e^{-i\pi/3} = 4(\cos(-\pi/3) + i\sin(-\pi/3)) = 4(1/2 - i\sqrt{3}/2) = 2 - 2\sqrt{3}i$

$z^4 = (z^2)^2 = (2 - 2\sqrt{3}i)^2 = 4 - 8\sqrt{3}i + 12 = 16 - 8\sqrt{3}i$

Substitute into $z^4 + 16(1+i)z^2 + a + bi = 0$:

$(16 - 8\sqrt{3}i) + 16(1+i)(2 - 2\sqrt{3}i) + a + bi = 0$

$= 16 - 8\sqrt{3}i + 32 + 32i - 32\sqrt{3}i - 32\sqrt{3}i^2 + a + bi$

$= (48 + 32 + 32\sqrt{3}) + i(-8\sqrt{3} + 32 - 32\sqrt{3}) + a + bi$

Wait, let me recalculate $16(1+i)(2-2\sqrt{3}i)$: $= 16[(2-2\sqrt{3}i) + i(2-2\sqrt{3}i)] = 16[2 - 2\sqrt{3}i + 2i - 2\sqrt{3}i^2]$ $= 16[2 + 2i - 2\sqrt{3}i + 2\sqrt{3}] = 32 + 32i - 32\sqrt{3}i + 32\sqrt{3}$

Total: $16 + 32 + 32\sqrt{3} + (-8\sqrt{3} + 32 - 32\sqrt{3})i + a + bi = 0$

Real: $48 + 32\sqrt{3} + a = 0$ ‚Üí $a = -48 - 32\sqrt{3}$

Imag: $-8\sqrt{3} + 32 + b = 0$ ‚Üí $b = 8\sqrt{3} - 32$

‚ö†Ô∏è Nuance: Since coefficients are real, complex roots come in conjugate pairs.

Common Patterns

De Moivre's Theorem:

$(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$

nth Roots of z:

If $z = re^{i\theta}$, then roots are $r^{1/n}e^{i(\theta + 2k\pi)/n}$ for $k = 0, 1, ..., n-1$

Trigonometric Identities:

$\cos n\theta = \sum_{k=0}^{\lfloor n/2 \rfloor} \binom{n}{2k} \cos^{n-2k}(-\sin)^{2k}\theta$
