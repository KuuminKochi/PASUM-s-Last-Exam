Question 2 [5 marks]

Capacitor with Teflon dielectric (k=2.1), area 0.50 m¬≤, C = 0.10 ŒºF. Find: - (a) thickness of Teflon - (b) charge at 12V

Need a hint?

Use $C = k\varepsilon_0 A/d$ to find d. Then $Q = CV$.

Show solution (a) $d = \frac{k\varepsilon_0 A}{C} = \frac{(2.1)(8.85 \times 10^{-12})(0.50)}{1.0 \times 10^{-7}} = 9.29 \times 10^{-6}\,$m = 9.29 Œºm (b) $Q = CV = (1.0 \times 10^{-7})(12) = 1.2 \times 10^{-6}\,$C = 1.2 ŒºC

‚ö†Ô∏è Nuance: Dielectric constant k increases capacitance by factor k.

Question 4 [5 marks]

Three capacitors (2ŒºF, 3ŒºF, 4ŒºF) in series with 6V battery. Find V across 4ŒºF capacitor.

Need a hint?

In series, same Q on each capacitor. Find C_eq, then Q = C_eq √ó V, then V = Q/C.

Show solution

$1/C_{eq} = 1/2 + 1/3 + 1/4 = 13/12$ ‚áí $C_{eq} = 12/13$ ŒºF

$Q = C_{eq}V = (12/13 \times 10^{-6})(6) = 72/13 \times 10^{-6}$ C

$V_{4\mu F} = Q/C = (72/13 \times 10^{-6})/(4 \times 10^{-6}) = 18/13 = 1.38$ V

‚ö†Ô∏è Nuance: In series, voltage divides inversely proportional to capacitance.

Common Patterns
