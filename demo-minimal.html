Tutorial 1: Integration Techniques - Part 1

Topics: Basic Integration, Substitution | Difficulty: Beginner | 22 questions ‚Ä¢ 100 marks ‚Ä¢ ~60 minutes

Question 1 [60 marks]

Find the indefinite integrals for the following functions.

Part (a) [5 marks]

$$\int (3e^{4x} - 5e^{-6x}) \, dx$$

Need a hint?

Split using linearity: $\int (f - g) = \int f - \int g$. Remember $\int e^{ax} dx = \frac{1}{a}e^{ax} + C$

Show solution

Method: Direct integration

$$\begin{aligned} \int (3e^{4x} - 5e^{-6x}) dx &= \frac{3e^{4x}}{4} - \frac{5e^{-6x}}{-6} + C \\ &= \boxed{\frac{3}{4}e^{4x} + \frac{5}{6}e^{-6x} + C} \end{aligned}$$

‚ö†Ô∏è Nuance: Watch the sign! $-\frac{5}{-6} = +\frac{5}{6}$

Part (b) [5 marks]

$$\int (6x^6 + 4x^3 + 8) \, dx$$

Need a hint?

Use power rule: $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ for each term

Show solution

$$\begin{aligned} \int (6x^6 + 4x^3 + 8) dx &= \frac{6x^7}{7} + \frac{4x^4}{4} + 8x + C \\ &= \boxed{\frac{6}{7}x^7 + x^4 + 8x + C} \end{aligned}$$

‚ö†Ô∏è Nuance: $\int x^3 dx = \frac{x^4}{4}$, not $\frac{4x^4}{4}$

Part (c) [5 marks]

$$\int (e^y + \sin y) \, dy$$

Need a hint?

$\int e^y dy = e^y$ and $\int \sin y dy = -\cos y$ (note the negative!)

Show solution

$$\int (e^y + \sin y) dy = \boxed{e^y - \cos y + C}$$

‚ö†Ô∏è Nuance: Don't forget: $\int \sin y \, dy = -\cos y$ (negative sign!)

Part (d) [5 marks]

$$\int 4a^2 e^x \, dx$$

Need a hint?

$4a^2$ is a constant (assuming $a$ is constant). Only $e^x$ gets integrated.

Show solution

$$\int 4a^2 e^x dx = 4a^2 \int e^x dx = \boxed{4a^2 e^x + C}$$

‚ö†Ô∏è Nuance: $a$ is treated as a constant coefficient

Part (e) [5 marks]

$$\int (2\tan x + 3\sec x) \, dx$$

Need a hint?

Standard results: $\int \tan x \, dx = \ln|\sec x|$ and $\int \sec x \, dx = \ln|\sec x + \tan x|$

Show solution

$$\begin{aligned} \int (2\tan x + 3\sec x) dx &= 2\ln|\sec x| + 3\ln|\sec x + \tan x| + C \end{aligned}$$

‚ö†Ô∏è Nuance: Memorize these standard trigonometric integrals

Part (f) [5 marks]

$$\int \frac{\sin^2 x + \cos^2 x}{3x} \, dx$$

Need a hint?

Simplify numerator first. What is $\sin^2 x + \cos^2 x$?

Show solution

$$\begin{aligned} \int \frac{\sin^2 x + \cos^2 x}{3x} dx &= \int \frac{1}{3x} dx \quad (\text{since } \sin^2 x + \cos^2 x = 1) \\ &= \frac{1}{3}\ln|x| + C \end{aligned}$$

‚ö†Ô∏è Nuance: Always simplify before integrating! Many miss this identity.

Part (g) [5 marks]

$$\int (4 + 2x)^5 \, dx$$

Need a hint?

Use reverse chain rule or substitution: $u = 4 + 2x$, so $du = 2\,dx$

Show solution

$$\int (4 + 2x)^5 dx = \frac{(4 + 2x)^6}{6 \cdot 2} + C = \boxed{\frac{(4 + 2x)^6}{12} + C}$$

‚ö†Ô∏è Nuance: Divide by derivative of inner function (which is 2)

Part (h) [5 marks]

$$\int 4e^4 \, dx$$

Need a hint?

$e^4$ is just a number (~54.6). It's NOT $e^{4x}$.

Show solution

$$\int 4e^4 dx = 4e^4 \int 1 \, dx = \boxed{4e^4 x + C}$$

‚ö†Ô∏è Nuance: Common mistake: trying to integrate $e^4$ as $e^{4x}$

Part (i) [5 marks]

$$\int (x + \sec x \tan x) \, dx$$

Show solution

$$\int (x + \sec x \tan x) dx = \boxed{\frac{x^2}{2} + \sec x + C}$$

‚ö†Ô∏è Nuance: $\int \sec x \tan x \, dx = \sec x$ is a standard result

Part (j) [5 marks]

$$\int \frac{(xe^{-2x} - 3x)e^x}{2x} \, dx$$

Need a hint?

Factor and simplify: $\frac{x(e^{-2x} - 3)e^x}{2x} = \frac{e^{-x} - 3e^x}{2}$

Show solution

$$\begin{aligned} \int \frac{(xe^{-2x} - 3x)e^x}{2x} dx &= \frac{1}{2}\int (e^{-x} - 3e^x) dx \\ &= \frac{1}{2}(-e^{-x} - 3e^x) + C \end{aligned}$$

‚ö†Ô∏è Nuance: Simplify algebraically first! Don't rush to integrate.

Part (k) [5 marks]

$$\int \frac{3x^3 - 5x}{x^7} \, dx$$

Need a hint?

Split: $\frac{3x^3}{x^7} - \frac{5x}{x^7} = 3x^{-4} - 5x^{-6}$

Show solution

$$\begin{aligned} \int \frac{3x^3 - 5x}{x^7} dx &= \int (3x^{-4} - 5x^{-6}) dx \\ &= \frac{3x^{-3}}{-3} - \frac{5x^{-5}}{-5} + C = -\frac{1}{x^3} + \frac{1}{x^5} + C \end{aligned}$$

‚ö†Ô∏è Nuance: Watch signs: $-\frac{5}{-5} = +1$

Part (l) [5 marks]

$$\int 5^{7x} \, dx$$

Need a hint?

Use formula: $\int a^{kx} dx = \frac{a^{kx}}{k\ln a} + C$. Don't forget $\ln a$!

Show solution

$$\int 5^{7x} dx = \boxed{\frac{5^{7x}}{7\ln 5} + C}$$

‚ö†Ô∏è Nuance: Remember the $\ln a$ in denominator: $\int a^x dx = \frac{a^x}{\ln a}$

Question 2 [40 marks]

Use appropriate substitution to find:

Part (a) [4 marks]

$$\int 3x^2(x^3 - 2)^3 \, dx$$

Need a hint?

Let $u = x^3 - 2$, then $du = 3x^2 dx$

Show solution

$$\begin{aligned} \text{Let } u = x^3 - 2, \, du = 3x^2 dx \\ \int 3x^2(x^3-2)^3 dx = \int u^3 du = \frac{u^4}{4} + C = \boxed{\frac{(x^3-2)^4}{4} + C} \end{aligned}$$

‚ö†Ô∏è Nuance: Look for pattern: $f(g(x)) \cdot g'(x)$

Part (b) [4 marks]

$$\int \cos x \sin^2 x \, dx$$

Need a hint?

Let $u = \sin x$, then $du = \cos x \, dx$

Show solution

$$\begin{aligned} \text{Let } u = \sin x, \, du = \cos x \, dx \\ \int \cos x \sin^2 x \, dx = \int u^2 du = \frac{u^3}{3} + C = \boxed{\frac{\sin^3 x}{3} + C} \end{aligned}$$

Common Patterns Identified

Pattern 1: Sign Errors with Exponentials ‚Äî High frequency (Q1a, Q1j)
Strategy: Write $\frac{1}{-a} = -\frac{1}{a}$ explicitly

Pattern 2: Missing Algebraic Simplification ‚Äî High frequency (Q1f, Q1j)
Strategy: Always simplify before integrating

Pattern 3: Substitution Recognition ‚Äî High frequency (Q2)
Strategy: Look for $f(g(x)) \cdot g'(x)$ structure
