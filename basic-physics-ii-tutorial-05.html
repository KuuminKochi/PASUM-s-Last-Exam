Tutorial 5: AC Circuits & Wheatstone Bridge

Question 1 [8 marks]

A voltage divider circuit is connected to a Wheatstone bridge network via a switch, Figure 1. Determine the: - (a) output voltage at Point A relative to the negative terminal of the power supply when the switch is open - (b) value of the unknown resistance $R_{\text{unknown}}$ when the switch is closed and the bridge is brought to equilibrium when variable resistor is $3.0 \, \Omega$ - (c) new output voltage at Point A, relative to the negative terminal of the power supply when the switch is closed and bridge is balanced

\begin{tikzpicture}[scale=1.1, >=stealth] % Voltage source \draw[thick] (0,1) -- (0,0); \node at (-0.5,0.5) {$V_{in}$}; \draw[thick] (1.5,1) -- (1.5,0); \draw (0,0) to[short, *-] (1.5,0); \draw (0,1) to[short, *-] (1.5,1);

% Voltage divider \draw (0,0.5) to[ resistor={$3\,\Omega$}] (1.5,0.5); \draw (0,0.5) to[ resistor={$6\,\Omega$}] (1.5,0.5);

% Point A \filldraw (0.75,0.5) circle (2pt) node[above] {$A$};

% Switch \draw (0.75,0.5) to[switch] (2.5,0.5);

% Wheatstone bridge \draw (2.5,0.5) -- (3,0.5); \draw (3,0.5) to[ resistor={$R_1$}] (3,2); \draw (3,2) to[ resistor={$R_2$}] (5,2); \draw (5,2) -- (5,0.5); \draw (5,0.5) to[ resistor={$R_3$}] (5,0); \draw (5,0) to[ resistor={$R_{unknown}$}] (3,0); \draw (3,0) -- (2.5,0.5);

% Labels \node at (0.75,-0.3) {Switch}; \node at (4,-0.3) {Wheatstone Bridge}; \node at (4,2.5) {Find: (a) $V_A$, (b) $R_{unknown}$, (c) $V_A$ when balanced}; \end{tikzpicture}

Need a hint?

For (a): Use voltage divider rule. For (b): In a balanced Wheatstone bridge, $R_1/R_2 = R_3/R_4$. For (c): Calculate voltage at Point A using the voltage divider with the new configuration after bridge is connected.

Show solution Given: Voltage divider with $3 \, \Omega$ and $6 \, \Omega$ resistors, Wheatstone bridge connected through switch. (a) Output voltage when switch is OPEN:

The voltage divider has total resistance $R_{\text{total}} = 3 + 6 = 9 \, \Omega$

If we assume the input voltage is across the $9 \, \Omega$ total:

$$I = \frac{V_{\text{in}}}{R_{\text{total}}} = \frac{V_{\text{in}}}{9}$$

The voltage across the $6 \, \Omega$ resistor (Point A):

$$V_A = I \times 6 = \frac{V_{\text{in}}}{9} \times 6 = \frac{2}{3}V_{\text{in}}$$

From the solution: $V_A = 6 \, \text{V}$ (assuming some input voltage)

(b) Unknown resistance when bridge is balanced:

For a balanced Wheatstone bridge: $\frac{R_1}{R_2} = \frac{R_3}{R_4}$

Looking at the circuit configuration with variable resistor $R_v = 3 \, \Omega$:

$$R_{\text{unknown}} = R_v \times \frac{R_1}{R_2}$$

From the solution:

$$R_{\text{unknown}} = 4.5 \, \Omega$$ (c) New output voltage when switch is CLOSED and bridge balanced:

The output voltage at Point A now depends on the parallel/series combination through the bridge.

Using the voltage divider formula with the effective resistance:

$$V_{\text{out}} = \frac{R_{\text{eff}}}{R_{\text{total}}} \times V_{\text{in}}$$

From the solution: $V_A = 4.15 \, \text{V}$ when the bridge is connected and balanced.

‚ö†Ô∏è Nuance: In a Wheatstone bridge, the output voltage is zero (or the bridge is balanced) when the ratio of resistances on opposite arms are equal. When unbalanced, the output can be found using superposition or Thevenin's theorem. The voltage divider provides a reference voltage, and the bridge compares this to the voltage across another portion of the circuit.

Question 2 [5 marks]

The current in an AC circuit is given by:

$$I(t) = 12 \sin(100\pi t)$$

Determine the: - (a) peak current - (b) rms current - (c) current when $t = 5 \, \text{ms}$ - (d) frequency and period of the current

\begin{tikzpicture}[scale=1.2, >=stealth] % AC waveform \draw[->] (0,0) -- (5,0) node[right] {$t$}; \draw[->] (0,-1.5) -- (0,1.5) node[above] {$I(t)$};

% Sine wave \draw[thick, red, domain=0:4.5] plot (\x, {1.2sin(\x r180/pi)});

% Labels \node at (2.5,1.8) {$I(t) = 12\sin(100\pi t)$}; \node at (4,-0.3) {$T = 0.02\,$s};

% Amplitude \draw[<->] (0.3,0) -- (0.3,1.2) node[midway,left] {$12\,$A}; \draw[dashed] (0,1.2) -- (0.3,1.2);

% Period \draw[<->] (1,-0.3) -- (1+0.4,-0.3) node[midway,below] {$T$}; \end{tikzpicture}

Need a hint?

For (a): Peak is the amplitude of the sine function. For (b): $I_{\text{rms}} = I_{\text{peak}}/\sqrt{2}$. For (c): Substitute t into the equation. For (d): Compare with standard form $I = I_0 \sin(\omega t)$ where $\omega = 2\pi f$.

Show solution Given: $I(t) = 12 \sin(100\pi t)$ (a) Peak current ($I_0$):

The coefficient of sine is the amplitude:

$$I_0 = 12 \, \text{A}$$ (b) RMS current: $$I_{\text{rms}} = \frac{I_0}{\sqrt{2}} = \frac{12}{\sqrt{2}} = 8.49 \, \text{A}$$ (c) Current at $t = 5 \, \text{ms}$:

Convert time: $t = 5 \, \text{ms} = 5 \times 10^{-3} \, \text{s}$

$$I(5 \text{ms}) = 12 \sin(100\pi \times 5 \times 10^{-3}) = 12 \sin(0.5\pi) = 12 \sin(90^\circ) = 12 \times 1 = 12 \, \text{A}$$ (d) Frequency and period:

Compare with standard form $I = I_0 \sin(\omega t)$:

$$\omega = 100\pi \, \text{rad/s}$$

Since $\omega = 2\pi f$:

$$f = \frac{\omega}{2\pi} = \frac{100\pi}{2\pi} = 50 \, \text{Hz}$$ $$T = \frac{1}{f} = \frac{1}{50} = 0.02 \, \text{s} = 20 \, \text{ms}$$

‚ö†Ô∏è Nuance: Peak current is the maximum instantaneous value. RMS current gives the effective DC equivalent that would produce the same heating effect. Note that at $t = 5$ ms (which is 1/4 of the period), the sine reaches its maximum value of 1.

Question 3 [5 marks]

An AC power supply delivers an rms current of $5 \, \text{A}$ at a frequency of $50 \, \text{Hz}$ to a resistor of $8 \, \Omega$. - (a) Calculate the peak current - (b) Determine the peak voltage across the resistor - (c) Write the expression for the instantaneous voltage

Need a hint?

For (a): $I_0 = I_{\text{rms}} \times \sqrt{2}$. For (b): Use Ohm's law $V_0 = I_0 R$. For (c): $v(t) = V_0 \sin(\omega t)$ where $\omega = 2\pi f$.

Show solution Given:

- $I_{\text{rms}} = 5 \, \text{A}$ - Frequency: $f = 50 \, \text{Hz}$ - Resistance: $R = 8 \, \Omega$

(a) Peak current: $$I_0 = I_{\text{rms}} \sqrt{2} = 5 \times 1.414 = 7.07 \, \text{A}$$ (b) Peak voltage: $$V_0 = I_0 R = 7.07 \times 8 = 56.6 \, \text{V}$$ (c) Instantaneous voltage expression:

Angular frequency:

$$\omega = 2\pi f = 2\pi(50) = 100\pi \, \text{rad/s}$$

The voltage expression:

$$v(t) = V_0 \sin(\omega t) = 56.6 \sin(100\pi t) \, \text{V}$$

Or more precisely:

$$v(t) = 56.6 \sin(100\pi t) \, \text{V}$$

‚ö†Ô∏è Nuance: For a purely resistive AC circuit, voltage and current are in phase (phase difference = 0). The voltage also follows a sine wave with the same frequency as the current. The peak voltage is $\sqrt{2}$ times the RMS voltage.

Need a hint?

A phasor rotating counterclockwise represents a sine wave. The length of the phasor = amplitude. The angle at t=0 determines the phase. If V leads I by angle œÜ, then V = V‚ÇÄ sin(œât + œÜ) and I = I‚ÇÄ sin(œât).

Show solution (a) Voltage leads current by 90¬∞ ($\phi = \pi/2$):

Voltage reaches maximum first (at t = T/4), then current reaches maximum. The voltage wave is shifted left compared to current.

(b) Current leads voltage by 90¬∞:

Current reaches maximum first, voltage follows. The current wave is shifted left compared to voltage.

(c) Voltage leads current by 45¬∞ ($\phi = \pi/4$):

Voltage peaks before current by T/8. Small phase difference.

(d) Current leads voltage by 45¬∞:

Current peaks before voltage by T/8.

‚ö†Ô∏è Nuance: Phasors rotate counterclockwise. At t=0, the phasor makes an angle with the horizontal axis equal to the phase. The projection on the vertical axis gives the instantaneous value. The sinusoidal waveform is the "snapshot" of this rotating phasor over one period.

Question 5 [5 marks]

The alternating current and voltage in a circuit are given by:

$$I(t) = 4 \cos \omega t$$ $$V(t) = 10 \sin \omega t$$

- (a) Determine the phase difference between the current and voltage - (b) State clearly which quantity leads and which lags - (c) Draw the phasor diagram showing the correct phase relationship between current and voltage

\begin{tikzpicture}[scale=1.3, >=stealth] % Phasor diagram at t = 0 \draw[->] (0,0) -- (3,0) node[right] {Real axis}; \draw[->] (0,0) -- (0,2.5) node[above] {Imaginary axis};

% Current phasor (at 90 degrees, from cos) \draw[->, thick, blue] (0,0) -- (0,2) node[above] {$I = 4\cos\omega t = 4\sin(\omega t + 90^\circ)$};

% Voltage phasor (at 0 degrees, from sin) \draw[->, thick, red] (0,0) -- (2.5,0) node[right] {$V = 10\sin\omega t$};

% Phase angle \draw[blue] (0.4,0) arc (0:90:0.4); \node[blue] at (0.6,0.5) {$90^\circ$};

% Labels \node at (1.5,-1) {Current LEADS Voltage by $90^\circ$}; \node at (1.5,-1.5) {(or Voltage LAGS Current by $90^\circ$)};

% Waveform sketch \begin{scope}[xshift=5cm] \draw[->] (0,0) -- (3,0) node[right] {$t$}; \draw[->] (0,-1.5) -- (0,1.5) node[above] {}; \draw[thick, blue, domain=0:2.5] plot (\x, {1.2cos(\x r180/pi)}); \draw[thick, red, domain=0:2.5] plot (\x, {1sin(\x r180/pi)}); \node[blue] at (2,1.3) {$I(t)$}; \node[red] at (2,-1.3) {$V(t)$}; \end{scope} \end{tikzpicture}

Need a hint?

For (a): Both need to be in the same trig function. Note that $\cos\theta = \sin(\theta + \pi/2)$, so $\cos\omega t = \sin(\omega t + \pi/2)$. For (b): The one with the larger angle leads. For (c): Draw phasors with correct angle difference.

Show solution (a) Phase difference:

First, express both in the same form. Using $\cos\theta = \sin(\theta + \pi/2)$:

$$I(t) = 4 \cos\omega t = 4 \sin\left(\omega t + \frac{\pi}{2}\right)$$ $$V(t) = 10 \sin\omega t$$

Now we can compare phases: - Current has phase: $+\frac{\pi}{2}$ (90¬∞) - Voltage has phase: $0$

Phase difference: $\phi = \frac{\pi}{2} - 0 = \frac{\pi}{2}$ (or 90¬∞)

(b) Which leads and which lags:

Since the current has a larger phase angle ($\frac{\pi}{2}$ vs $0$), current leads voltage by 90¬∞.

Equivalently, voltage lags current by 90¬∞.

(c) Phasor diagram:

At t = 0, the current phasor points at 90¬∞ (vertical up), while the voltage phasor points at 0¬∞ (horizontal right). Since phasors rotate counterclockwise, the current reaches its maximum first - this confirms current leads voltage by 90¬∞.

‚ö†Ô∏è Nuance: When current leads voltage, the current reaches its peak earlier (by $\phi/\omega$ seconds). This happens in circuits with capacitive elements. When voltage leads current (inductive circuits), the current lags. The phase relationship tells us about the nature of the circuit.

Common Patterns

RMS Values:

- $I_{\text{rms}} = I_0/\sqrt{2}$ - $V_{\text{rms}} = V_0/\sqrt{2}$ - $P_{\text{avg}} = V_{\text{rms}} I_{\text{rms}} = I_{\text{rms}}^2 R$

Angular Frequency:

- $\omega = 2\pi f$ - $T = 1/f$ - $f = \omega/2\pi$

Instantaneous Values:

- $i(t) = I_0 \sin(\omega t + \phi)$ - $v(t) = V_0 \sin(\omega t + \phi_V)$

Phase Relationships:

- If $\phi > 0$: quantity leads - If $\phi < 0$: quantity lags - Phase difference $\Delta\phi = \phi_V - \phi_I$

Wheatstone Bridge:

- Balanced: $R_1/R_2 = R_3/R_4$ ‚Üí $V_{\text{out}} = 0$ - Unbalanced: Use voltage divider or mesh analysis

Voltage Divider:

- $V_{\text{out}} = \frac{R_2}{R_1 + R_2} \times V_{\text{in}}$
