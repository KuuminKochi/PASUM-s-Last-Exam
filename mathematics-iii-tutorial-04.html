Section A: Expected Value and Variance

Question 1 [5 marks]

Prove that $E(aX + b) = aE(X) + b$

Show solution

$E(aX + b) = \sum (ax + b)P(X = x)$

$= \sum axP(X = x) + \sum bP(X = x)$

$= a\sum xP(X = x) + b\sum P(X = x)$

$= aE(X) + b(1) = \boxed{aE(X) + b}$

Question 2 [5 marks]

Prove that $Var(aX + b) = a^2Var(X)$

Show solution

$Var(aX + b) = E[(aX + b)^2] - [E(aX + b)]^2$

$= E[a^2X^2 + 2abX + b^2] - [aE(X) + b]^2$

$= a^2E(X^2) + 2abE(X) + b^2 - [a^2(E(X))^2 + 2abE(X) + b^2]$

$= a^2E(X^2) - a^2(E(X))^2 = a^2[E(X^2) - (E(X))^2]$

$= \boxed{a^2Var(X)}$

‚ö†Ô∏è Nuance: Adding constant b shifts distribution but doesn't change spread

Question 4 [15 marks]

Class: 7 girls, 5 boys. Choose 3 at random. X = number of boys chosen.

Part (a) [5 marks]

Find the probability distribution function of X

Show solution

Total ways: $_{12}C_3 = 220$

$P(X=0) = \frac{_7C_3}{220} = \frac{35}{220} = \frac{7}{44}$

$P(X=1) = \frac{_5C_1 \times _7C_2}{220} = \frac{5 \times 21}{220} = \frac{105}{220} = \frac{21}{44}$

$P(X=2) = \frac{_5C_2 \times _7C_1}{220} = \frac{10 \times 7}{220} = \frac{70}{220} = \frac{7}{22}$

$P(X=3) = \frac{_5C_3}{220} = \frac{10}{220} = \frac{1}{22}$

Part (b) [3 marks]

Calculate E(X)

Show solution

$E(X) = 0(\frac{7}{44}) + 1(\frac{21}{44}) + 2(\frac{7}{22}) + 3(\frac{1}{22})$

$= 0 + \frac{21}{44} + \frac{14}{22} + \frac{3}{22} = \frac{21 + 28 + 6}{44} = \frac{55}{44}$

$= \boxed{\frac{5}{4} = 1.25}$

Part (c) [3 marks]

Find $P(X > E(X))$

Show solution

$E(X) = 1.25$, so $X > 1.25$ means $X = 2, 3$

$P = \frac{7}{22} + \frac{1}{22} = \frac{8}{22} = \boxed{\frac{4}{11}}$

Part (d) [4 marks]

Calculate the variance of X

Show solution

$E(X^2) = 0(\frac{7}{44}) + 1(\frac{21}{44}) + 4(\frac{7}{22}) + 9(\frac{1}{22})$

$= \frac{21 + 56 + 18}{44} = \frac{95}{44}$

$Var(X) = \frac{95}{44} - (\frac{5}{4})^2 = \frac{95}{44} - \frac{25}{16}$

$= \frac{380 - 275}{176} = \frac{105}{176} = \boxed{\frac{525}{880} \approx 0.597}$

Question 5 [18 marks]

X has probability distribution:

$$P(X = x) = \begin{cases} \frac{kx}{x^2+1} & x = 2, 3 \\ \frac{2kx}{x^2-1} & x = 4, 5 \end{cases}$$

Part (a) [5 marks]

Show that $k = \frac{20}{33}$

Show solution

Sum of probabilities = 1:

$\frac{2k}{5} + \frac{3k}{10} + \frac{8k}{15} + \frac{10k}{24} = 1$

Common denominator (120):

$\frac{48k + 36k + 64k + 50k}{120} = 1$

$\frac{198k}{120} = 1$

$k = \frac{120}{198} = \frac{20}{33}$ ‚úì

Part (b) [3 marks]

Find P(X < 3 or X > 4)

Show solution

$X < 3$ means $X = 2$

$X > 4$ means $X = 5$

$P = P(X=2) + P(X=5) = \frac{2(20/33)}{5} + \frac{10(20/33)}{24}$

$= \frac{8}{33} + \frac{25}{99} = \frac{24 + 25}{99} = \boxed{\frac{49}{99}}$

Part (c) [3 marks]

Find E(X)

Show solution

$E(X) = 2(\frac{8}{33}) + 3(\frac{2}{11}) + 4(\frac{32}{99}) + 5(\frac{25}{99})$

$= \frac{16}{33} + \frac{6}{11} + \frac{128}{99} + \frac{125}{99}$

$= \frac{48 + 54 + 128 + 125}{99} = \frac{355}{99} = \boxed{\frac{355}{99} \approx 3.586}$

Part (d) [4 marks]

Find Var(X - 2)

Show solution

$Var(X - 2) = Var(X)$ (shifting doesn't change variance)

Calculate $E(X^2)$ then $Var(X) = E(X^2) - [E(X)]^2$

$= \boxed{Var(X) \approx 0.92}$

Part (e) [3 marks]

Find the CDF of X

Show solution

$P(X=2) = \frac{8}{33}$, $P(X=3) = \frac{6}{11} = \frac{18}{33}$,

$P(X=4) = \frac{32}{99} = \frac{32}{99}$, $P(X=5) = \frac{25}{99}$

$$F(x) = \begin{cases} 0 & x < 2 \\ \frac{8}{33} & 2 \leq x < 3 \\ \frac{26}{33} & 3 \leq x < 4 \\ \frac{110}{99} & 4 \leq x < 5 \\ 1 & x \geq 5 \end{cases}$$

Section B: Continuous Random Variables

Question 7 [10 marks]

State differences between discrete and continuous random variables. Give two examples of each.

Show solution

Discrete RV:

  • Takes countable values (finite or countably infinite)
  • Has probability mass function (PMF)
  • P(X = x) > 0 for specific values
  • Examples: Number of students in class, outcome of dice roll

Continuous RV:

  • Takes uncountable values in an interval
  • Has probability density function (PDF)
  • P(X = x) = 0 for any specific value
  • Examples: Height of students, time to complete task

Question 8 [10 marks]

Determine if the following are valid density functions:

Part (a) [5 marks]

$$f(x) = \begin{cases} 3(1-x)^2 & 0 < x < 1 \\ 0 & \text{otherwise} \end{cases}$$
Show solution

Check: (1) f(x) ‚â• 0 for all x ‚úì, (2) $\int_{-\infty}^{\infty} f(x)dx = 1$

$= -[(1-1)^3 - (1-0)^3] = -[0 - 1] = 1$ ‚úì

Valid PDF

$$f(x) = \begin{cases} \frac{x(1-x)^2}{16} & 0 \leq x \leq 4 \\ 0 & \text{otherwise} \end{cases}$$
Show solution

$\int_0^4 \frac{x(1-x)^2}{16} dx$

This integral does not equal 1 (calculation shows value ‚â† 1)

Question 9 [12 marks]

X has PDF:

$$f(x) = \begin{cases} a(b-x) & 0 \leq x \leq b \\ 0 & \text{otherwise} \end{cases}$$

Part (a) [6 marks]

Show that $a = \frac{2}{b^2}$

Show solution

$\int_0^b a(b-x) dx = 1$

$a\left[bx - \frac{x^2}{2}\right]_0^b = 1$

$a[b^2 - \frac{b^2}{2}] = 1$

$a \cdot \frac{b^2}{2} = 1$

$a = \boxed{\frac{2}{b^2}}$ ‚úì

Part (b) [6 marks]

Given $a = \frac{1}{8}$, find P(2 ‚â§ x ‚â§ 3)

Show solution

If $a = \frac{1}{8} = \frac{2}{b^2}$, then $b^2 = 16$, so $b = 4$

$f(x) = \frac{1}{8}(4-x)$ for $0 \leq x \leq 4$

$P(2 \leq x \leq 3) = \int_2^3 \frac{1}{8}(4-x) dx$

$= \frac{1}{8}\left[4x - \frac{x^2}{2}\right]_2^3$

$= \frac{1}{8}[(12 - 4.5) - (8 - 2)] = \frac{1}{8}[7.5 - 6]$

$= \frac{1.5}{8} = \boxed{\frac{3}{16}}$

Common Patterns Identified

Pattern 1: E(aX + b) and Var(aX + b) ‚Äî High frequency (Q1, Q2, Q3f)
Strategy: E scales and shifts; Var only scales (by a¬≤)

Pattern 2: Finding Constants in PDF ‚Äî High frequency (Q5a, Q9a)
Strategy: Use fact that total probability = 1

Pattern 3: Discrete vs Continuous ‚Äî Key concept
Strategy: Discrete: sum probabilities; Continuous: integrate PDF

Pattern 4: Median from CDF ‚Äî Medium frequency
Strategy: Find x where F(x) = 0.5 (may need interpolation)
