Tutorial 2: Integration by Parts

Overview

This tutorial covers integration by parts (IBP) technique, reduction formulas, and integration of logarithmic functions.

Topics: Integration by Parts, Reduction Formula Difficulty: Medium Estimated Time: 75 minutes

---

Question 1 [30 marks]

Part (a) [5 marks]

Show that $\int x^m e^{ax} dx = \frac{1}{a}x^m e^{ax} - \frac{m}{a}\int x^{m-1}e^{ax} dx$

Show solution

Method: Integration by parts with $u = x^m$, $dv = e^{ax}dx$

$du = mx^{m-1}dx$, $v = \frac{1}{a}e^{ax}$

$$\int x^m e^{ax} dx = \frac{1}{a}x^m e^{ax} - \int \frac{m}{a}x^{m-1}e^{ax} dx$$

$$= \frac{1}{a}x^m e^{ax} - \frac{m}{a}\int x^{m-1}e^{ax} dx$$

‚ö†Ô∏è Nuance: This is a reduction formula - it reduces the power of x each time

Part (b) [8 marks]

Hence, evaluate:

(i) [3 marks]

$$\int x^3 e^{2x} dx$$
Show solution

Apply reduction formula repeatedly:

$\int x^3 e^{2x} dx = \frac{1}{2}x^3 e^{2x} - \frac{3}{2}\int x^2 e^{2x} dx$

$= \frac{1}{2}x^3 e^{2x} - \frac{3}{2}\left[\frac{1}{2}x^2 e^{2x} - \int x e^{2x} dx\right]$

Continue until power of x is 0...

Answer: $\boxed{\frac{1}{2}x^3e^{2x} - \frac{3}{4}x^2e^{2x} + \frac{3}{4}xe^{2x} - \frac{3}{8}e^{2x} + C}$

(ii) [3 marks]

$$\int x^2(e^x - 1) dx$$
Show solution

Split: $\int x^2 e^x dx - \int x^2 dx$

Second part: $\frac{x^3}{3}$

First part by IBP: $x^2e^x - 2xe^x + 2e^x$

Answer: $\boxed{x^2e^x - 2xe^x + 2e^x - \frac{x^3}{3} + C}$

(iii) [2 marks]

$$\int x e^x dx$$
Show solution

$u = x$, $dv = e^x dx$

$= xe^x - \int e^x dx = \boxed{xe^x - e^x + C}$

Question 2 [30 marks]

Integrate the following:

Part (a) [5 marks]

$$\int \ln x \, dx$$
Show solution

$u = \ln x$, $dv = dx$

$du = \frac{1}{x}dx$, $v = x$

$= x\ln x - \int x \cdot \frac{1}{x} dx = x\ln x - \int 1 \, dx$

$= \boxed{x\ln x - x + C}$

‚ö†Ô∏è Nuance: Classic IBP problem - remember this result!

Part (b) [5 marks]

$$\int \ln x^2 \, dx$$
Show solution

Simplify first: $\ln x^2 = 2\ln x$

$= 2\int \ln x \, dx = \boxed{2x\ln x - 2x + C}$

Part (c) [5 marks]

$$\int (\ln x)^2 \, dx$$
Show solution

$u = (\ln x)^2$, $dv = dx$

$du = 2\ln x \cdot \frac{1}{x} dx$, $v = x$

$= x(\ln x)^2 - 2\int \ln x \, dx$

$= x(\ln x)^2 - 2(x\ln x - x) + C$

$= \boxed{x(\ln x)^2 - 2x\ln x + 2x + C}$

Part (d) [5 marks]

$$\int x^2 \sin x \, dx$$
Show solution

Apply IBP twice:

First: $u = x^2$, $dv = \sin x \, dx$

$= -x^2\cos x + 2\int x\cos x \, dx$

Second IBP: $u = x$, $dv = \cos x \, dx$

$= -x^2\cos x + 2(x\sin x + \cos x) + C$

$= \boxed{-x^2\cos x + 2x\sin x + 2\cos x + C}$

Part (e) [5 marks]

$$\int e^x \sin 2x \, dx$$
Show solution

Apply IBP twice and solve algebraically:

Let $I = \int e^x \sin 2x \, dx$

First IBP: $u = \sin 2x$, $dv = e^x dx$

$I = e^x\sin 2x - 2\int e^x \cos 2x \, dx$

Second IBP: $= e^x\sin 2x - 2[e^x\cos 2x + 2\int e^x\sin 2x \, dx]$

$I = e^x\sin 2x - 2e^x\cos 2x - 4I$

$5I = e^x(\sin 2x - 2\cos 2x)$

$I = \boxed{\frac{1}{5}e^x(\sin 2x - 2\cos 2x) + C}$

‚ö†Ô∏è Nuance: When IBP cycles back, solve algebraically!

Part (f) [5 marks]

$$\int x^3 e^x dx$$
Show solution

Apply IBP three times (or use reduction formula from Q1):

$= \boxed{x^3e^x - 3x^2e^x + 6xe^x - 6e^x + C}$

Question 3 [15 marks]

Identify what method(s) can be used to find $\int x\sqrt{3x+7} \, dx$, and then find it.

Show solution

Methods possible:

  • Method 1: Substitution $u = 3x + 7$
  • Method 2: Integration by Parts

Using substitution:

Let $u = 3x + 7$, then $du = 3dx$, $x = \frac{u-7}{3}$

$\int x\sqrt{3x+7} \, dx = \int \frac{u-7}{3} \cdot \sqrt{u} \cdot \frac{du}{3}$

$= \frac{1}{9}\int (u^{3/2} - 7u^{1/2}) du$

$= \frac{1}{9}\left[\frac{2}{5}u^{5/2} - \frac{14}{3}u^{3/2}\right] + C$

$= \boxed{\frac{2}{45}(3x+7)^{5/2} - \frac{14}{27}(3x+7)^{3/2} + C}$

Or simplified: $\frac{2}{135}(3x+7)^{3/2}(3x - 14) + C$

Question 4 [25 marks]

Solve using integration by parts:

Part (a) [6 marks]

$$\int x\sqrt{1+x} \, dx$$
Show solution

Let $u = x$, $dv = \sqrt{1+x} \, dx = (1+x)^{1/2} dx$

$v = \frac{2}{3}(1+x)^{3/2}$

$= x \cdot \frac{2}{3}(1+x)^{3/2} - \int \frac{2}{3}(1+x)^{3/2} dx$

$= \frac{2}{3}x(1+x)^{3/2} - \frac{2}{3} \cdot \frac{2}{5}(1+x)^{5/2} + C$

$= \boxed{\frac{2}{3}x(1+x)^{3/2} - \frac{4}{15}(1+x)^{5/2} + C}$

Part (b) [6 marks]

$$\int \frac{(\ln x)^2}{x} \, dx$$
Show solution

Substitution is easier: let $u = \ln x$, $du = \frac{1}{x}dx$

$= \int u^2 du = \frac{u^3}{3} + C = \boxed{\frac{(\ln x)^3}{3} + C}$

Part (c) [6 marks]

$$\int \ln(x^2 + 1) \, dx$$
Show solution

$u = \ln(x^2+1)$, $dv = dx$

$du = \frac{2x}{x^2+1}dx$, $v = x$

$= x\ln(x^2+1) - \int \frac{2x^2}{x^2+1} dx$

$= x\ln(x^2+1) - 2\int \left(1 - \frac{1}{x^2+1}\right) dx$

$= \boxed{x\ln(x^2+1) - 2x + 2\arctan x + C}$

Part (d) [7 marks]

$$\int e^x \cos x \, dx$$
Show solution

Let $I = \int e^x \cos x \, dx$

Apply IBP twice:

$I = e^x\cos x + \int e^x \sin x \, dx$

$= e^x\cos x + e^x\sin x - \int e^x \cos x \, dx$

$I = e^x(\cos x + \sin x) - I$

$2I = e^x(\cos x + \sin x)$

$I = \boxed{\frac{1}{2}e^x(\cos x + \sin x) + C}$

‚ö†Ô∏è Nuance: Classic cyclic IBP - solve algebraically for I

Common Patterns Identified

Pattern 1: LIATE Rule ‚Äî High frequency
Strategy: Choose u by priority: Log, Inverse trig, Algebraic, Trig, Exponential

Pattern 2: Cyclic IBP ‚Äî Medium frequency (Q2e, Q4d)
Strategy: When integral reappears, solve algebraically

Pattern 3: Reduction Formulas ‚Äî Medium frequency (Q1)
Strategy: Apply IBP repeatedly to reduce power

Pattern 4: Choosing dv ‚Äî High frequency
Strategy: dv should be easily integrable; sometimes substitution is better
