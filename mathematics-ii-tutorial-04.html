Tutorial 4: Trigonometric Substitution, Partial Fractions, and Definite Integrals

Overview

This tutorial covers advanced integration techniques including trigonometric substitution, partial fraction decomposition, and definite integrals.

Topics: Trigonometric Substitution, Partial Fractions, Definite Integrals Difficulty: Advanced Estimated Time: 90 minutes

---

Question 1 [30 marks]

Using appropriate trigonometric substitutions, find:

Part (a) [5 marks]

$$\int \frac{x}{\sqrt{x^2 - 25}} \, dx$$
Need a hint?

Use $x = 5\sec\theta$ for $\sqrt{x^2 - a^2}$ form

Show solution

Let $x = 5\sec\theta$, $dx = 5\sec\theta\tan\theta \, d\theta$

$\sqrt{x^2 - 25} = 5\tan\theta$

$= \int \frac{5\sec\theta}{5\tan\theta} \cdot 5\sec\theta\tan\theta \, d\theta$

$= 5\int \sec^2\theta \, d\theta = 5\tan\theta + C$

$= \boxed{\sqrt{x^2 - 25} + C}$

Part (b) [5 marks]

$$\int \frac{dx}{x^2\sqrt{4-x^2}}$$
Show solution

Let $x = 2\sin\theta$, $dx = 2\cos\theta \, d\theta$

$\sqrt{4-x^2} = 2\cos\theta$

$= \int \frac{2\cos\theta}{4\sin^2\theta \cdot 2\cos\theta} d\theta = \frac{1}{4}\int \csc^2\theta \, d\theta$

$= -\frac{1}{4}\cot\theta + C = \boxed{-\frac{\sqrt{4-x^2}}{4x} + C}$

Part (c) [5 marks]

$$\int \frac{dx}{x\sqrt{x^2+4}}$$
Show solution

Let $x = 2\tan\theta$, use substitution

$= \boxed{-\frac{1}{2}\ln\left|\frac{2+\sqrt{x^2+4}}{x}\right| + C}$

Part (d) [5 marks]

$$\int \frac{dx}{(4x^2-9)^{3/2}}$$
Show solution

Let $2x = 3\sec\theta$, so $x = \frac{3}{2}\sec\theta$

$(4x^2-9)^{3/2} = 27\tan^3\theta$

After substitution and simplification:

$= \boxed{-\frac{x}{9\sqrt{4x^2-9}} + C}$

Part (e) [5 marks]

$$\int \frac{x^3}{\sqrt{16-x^2}} \, dx$$
Show solution

Let $x = 4\sin\theta$

Or use algebraic substitution: $u = 16-x^2$, $du = -2x \, dx$

$= \boxed{-\frac{(16-x^2)^{3/2}}{3} + 16\sqrt{16-x^2} + C}$

Part (f) [5 marks]

$$\int \frac{dx}{x^2\sqrt{x^2+9}}$$
Show solution

Let $x = 3\tan\theta$

$= \boxed{-\frac{\sqrt{x^2+9}}{9x} + C}$

Question 2 [10 marks]

Given $\int \sec\alpha \, d\alpha = \ln|\sec\alpha + \tan\alpha| + C$, find:

$$\int \frac{1}{\sqrt{2x^2-4}} \, dx$$
Show solution

Rewrite: $\frac{1}{\sqrt{2}}\int \frac{dx}{\sqrt{x^2-2}}$

Let $x = \sqrt{2}\sec\theta$

$= \frac{1}{\sqrt{2}}\ln|x + \sqrt{x^2-2}| + C$

Or: $\boxed{\frac{1}{\sqrt{2}}\ln\left|\frac{x}{\sqrt{2}} + \sqrt{\frac{x^2}{2}-1}\right| + C}$

Question 3 [10 marks]

Evaluate the definite integral:

$$\int_0^4 |x^2 - 2| \, dx$$
Need a hint?

Find where $x^2 - 2 = 0$, split the integral at that point

Show solution

$x^2 - 2 = 0$ when $x = \sqrt{2}$

For $0 \leq x < \sqrt{2}$: $|x^2-2| = 2-x^2$

For $\sqrt{2} \leq x \leq 4$: $|x^2-2| = x^2-2$

$= \int_0^{\sqrt{2}}(2-x^2)dx + \int_{\sqrt{2}}^4(x^2-2)dx$

$= \left[2x - \frac{x^3}{3}\right]_0^{\sqrt{2}} + \left[\frac{x^3}{3} - 2x\right]_{\sqrt{2}}^4$

$= \left(2\sqrt{2} - \frac{2\sqrt{2}}{3}\right) + \left(\frac{64}{3}-8 - \frac{2\sqrt{2}}{3}+2\sqrt{2}\right)$

$= \boxed{\frac{40}{3} + \frac{4\sqrt{2}}{3}}$ or $\frac{40 + 4\sqrt{2}}{3}$

‚ö†Ô∏è Nuance: Always split absolute value integrals at the zero point

Question 4 [20 marks]

Evaluate using partial fractions:

Part (a) [10 marks]

$$\int \frac{dx}{(x+4)(x-1)}$$
Show solution

Partial fractions: $\frac{1}{(x+4)(x-1)} = \frac{A}{x+4} + \frac{B}{x-1}$

$1 = A(x-1) + B(x+4)$

At $x=1$: $1 = 5B$, so $B = \frac{1}{5}$

At $x=-4$: $1 = -5A$, so $A = -\frac{1}{5}$

$= \frac{1}{5}\int \left(\frac{1}{x-1} - \frac{1}{x+4}\right) dx$

$= \boxed{\frac{1}{5}\ln\left|\frac{x-1}{x+4}\right| + C}$

Part (b) [10 marks]

$$\int \frac{x+5}{x^2-1} \, dx$$
Show solution

$x^2-1 = (x+1)(x-1)$

$\frac{x+5}{(x+1)(x-1)} = \frac{A}{x+1} + \frac{B}{x-1}$

$x+5 = A(x-1) + B(x+1)$

At $x=1$: $6 = 2B$, $B = 3$

At $x=-1$: $4 = -2A$, $A = -2$

$= \int \left(\frac{-2}{x+1} + \frac{3}{x-1}\right) dx$

$= \boxed{-2\ln|x+1| + 3\ln|x-1| + C}$

Or: $\ln\left|\frac{(x-1)^3}{(x+1)^2}\right| + C$

Question 5 [15 marks]

Given $\int \frac{1}{x^2+1} dx = \tan^{-1}x + C$, find:

Part (a) [7 marks]

$$\int \frac{x^3-2x^2+3}{x^4+x^2} \, dx$$
Show solution

Factor: $x^4+x^2 = x^2(x^2+1)$

Partial fractions: $\frac{x^3-2x^2+3}{x^2(x^2+1)} = \frac{A}{x} + \frac{B}{x^2} + \frac{Cx+D}{x^2+1}$

Solve for A, B, C, D...

$= \boxed{\ln|x| + \frac{3}{x} - 2\tan^{-1}x + C}$

Part (b) [8 marks]

$$\int \frac{5x^3-4x^2+2x-3}{x^4+x^2} \, dx$$
Show solution

Similar approach with partial fractions

$= \boxed{\frac{3}{x} + 2\ln|x| + \frac{5}{2}\ln(x^2+1) - 4\tan^{-1}x + C}$

Question 6 [15 marks]

Evaluate the definite integrals:

Part (a) [3 marks]

$$\int_1^2 \left(x - \frac{1}{x}\right)\left(x + \frac{1}{x}\right) dx$$
Show solution

Expand: $\left(x - \frac{1}{x}\right)\left(x + \frac{1}{x}\right) = x^2 - \frac{1}{x^2}$

$= \left[\frac{x^3}{3} + \frac{1}{x}\right]_1^2 = \left(\frac{8}{3} + \frac{1}{2}\right) - \left(\frac{1}{3} + 1\right)$

$= \boxed{\frac{11}{6}}$

Part (b) [3 marks]

$$\int_0^3 (1-x)\sqrt{x} \, dx$$
Show solution

$= \int_0^3 (x^{1/2} - x^{3/2}) dx = \left[\frac{2}{3}x^{3/2} - \frac{2}{5}x^{5/2}\right]_0^3$

$= \frac{2}{3}(3\sqrt{3}) - \frac{2}{5}(9\sqrt{3}) = 2\sqrt{3} - \frac{18\sqrt{3}}{5}$

$= \boxed{-\frac{8\sqrt{3}}{5}}$

Part (c) [3 marks]

$$\int_1^2 (2-3x)^2 \, dx$$
Show solution

Expand: $4 - 12x + 9x^2$

$= \left[4x - 6x^2 + 3x^3\right]_1^2 = (8-24+24) - (4-6+3)$

$= 8 - 1 = \boxed{7}$

Part (d) [2 marks]

$$\int_0^{\pi} (\sin x + \cos x) \, dx$$
Show solution

$= \left[-\cos x + \sin x\right]_0^{\pi} = (1+0) - (-1+0)$

$= \boxed{2}$

Part (e) [2 marks]

$$\int_0^2 (e^x + 5^x) \, dx$$
Show solution

$= \left[e^x + \frac{5^x}{\ln 5}\right]_0^2 = (e^2 + \frac{25}{\ln 5}) - (1 + \frac{1}{\ln 5})$

$= \boxed{e^2 - 1 + \frac{24}{\ln 5}}$

Part (f) [2 marks]

$$\int_1^e \left(x - \frac{1}{x}\right) dx$$
Show solution

$= \left[\frac{x^2}{2} - \ln|x|\right]_1^e = \left(\frac{e^2}{2} - 1\right) - \left(\frac{1}{2} - 0\right)$

$= \boxed{\frac{e^2}{2} - \frac{3}{2}}$

Common Patterns Identified

Pattern 1: Trig Substitution Forms ‚Äî High frequency
Strategy: $\sqrt{a^2-x^2}$: use $x=a\sin\theta$ | $\sqrt{x^2-a^2}$: use $x=a\sec\theta$ | $\sqrt{x^2+a^2}$: use $x=a\tan\theta$

Pattern 2: Partial Fraction Cases ‚Äî High frequency
Strategy: Distinct linear: $\frac{A}{x-a} + \frac{B}{x-b}$ | Repeated: $\frac{A}{x} + \frac{B}{x^2}$ | Irreducible quadratic: $\frac{Ax+B}{x^2+c}$

Pattern 3: Absolute Value Definite Integrals ‚Äî Medium frequency (Q3)
Strategy: Always split at where expression inside absolute value equals zero

Pattern 4: Check for Simplification First ‚Äî High frequency
Strategy: Before diving into technique, check if algebra simplifies (Q6a)
