Tutorial 1: Integration Techniques - Part 1

Overview

This tutorial covers basic integration techniques including direct integration, exponential functions, trigonometric integrals, and introduction to substitution.

Topics: Basic Integration, Integration by Substitution Difficulty: Beginner Estimated Time: 60 minutes

---

Question 1 [60 marks]

Find the indefinite integrals for the following functions.

Part (a) [5 marks]\n$$\int (3e^{4x} - 5e^{-6x}) \, dx$$

Concepts: Exponential integration, linearity of integrals Key Formula: $\int e^{ax} \, dx = \frac{1}{a}e^{ax} + C$

Part (b) [5 marks]

$$\int (6x^6 + 4x^3 + 8) \, dx$$ Concepts: Power rule for integration

Part (c) [5 marks]

$$\int (e^y + \sin y) \, dy$$ Concepts: Variable substitution ($y$ instead of $x$), trigonometric integration

Part (d) [5 marks]

$$\int 4a^2 e^x \, dx$$ Concepts: Constants in integration ($a$ is a constant)

Part (e) [5 marks]

$$\int (2\tan x + 3\sec x) \, dx$$ Concepts: Trigonometric integrals

Part (f) [5 marks]

$$\int \frac{\sin^2 x + \cos^2 x}{3x} \, dx$$ Concepts: Trigonometric identity ($\sin^2 x + \cos^2 x = 1$), logarithmic integration

Part (g) [5 marks]

$$\int (4 + 2x)^5 \, dx$$ Concepts: Chain rule in reverse, substitution preparation

Part (h) [5 marks]

$$\int 4e^4 \, dx$$ Concepts: Constant integration (note: $e^4$ is a constant)

Part (i) [5 marks]

$$\int (x + \sec x \tan x) \, dx$$ Concepts: Standard integral $\int \sec x \tan x \, dx = \sec x + C$

Part (j) [5 marks]

$$\int \frac{(xe^{-2x} - 3x)e^x}{2x} \, dx$$ Concepts: Algebraic simplification before integration

Part (k) [5 marks]

$$\int \frac{3x^3 - 5x}{x^7} \, dx$$ Concepts: Simplifying rational functions, negative exponents

Part (l) [5 marks]

$$\int 5^{7x} \, dx$$ Concepts: Integration of $a^x$, using $a^x = e^{x\ln a}$

Question 2 [40 marks]

Use appropriate substitution to find:

Part (a) [4 marks]

$$\int 3x^2(x^3 - 2)^3 \, dx$$ Hint: Try $u = x^3 - 2$

Part (b) [4 marks]

$$\int \cos x \sin^2 x \, dx$$ Hint: Try $u = \sin x$

Part (c) [4 marks]

$$\int x\sqrt{x^2 + 1} \, dx$$ Hint: Try $u = x^2 + 1$

Part (d) [4 marks]

$$\int \frac{e^{\frac{1}{\mu}}}{\mu^2} \, d\mu$$ Hint: Try $u = \frac{1}{\mu}$

Part (e) [4 marks]

$$\int \frac{\sec^2 x}{(1 + \tan x)^3} \, dx$$ Hint: Try $u = 1 + \tan x$

Part (f) [4 marks]

$$\int x^2 \tan(x^3) \sec(x^3) \, dx$$ Hint: Try $u = x^3$

Part (g) [4 marks]

$$\int \frac{t^2 + 2t}{t^3 + 3t^2 + 10} \, dt$$ Hint: Try $u = t^3 + 3t^2 + 10$

Part (h) [4 marks]

$$\int \frac{(\ln x)^5}{x} \, dx$$ Hint: Try $u = \ln x$

Part (i) [4 marks]

$$\int \frac{\cos^3(4\theta)}{\sin^3(4\theta)} \, d\theta$$ Hint: Rewrite as $\cot^3(4\theta)$ or use substitution $u = \sin(4\theta)$

Part (j) [4 marks]

$$\int \frac{x^3 + 2x^2 + 5x + 6}{x + 2} \, dx$$ Hint: Polynomial long division first, or factor numerator

Quick Reference

Essential Integration Formulas

1. $\int x^n \, dx = \frac{x^{n+1}}{n+1} + C$ (for $n \neq -1$) 2. $\int e^x \, dx = e^x + C$ 3. $\int e^{ax} \, dx = \frac{1}{a}e^{ax} + C$ 4. $\int a^x \, dx = \frac{a^x}{\ln a} + C$ 5. $\int \frac{1}{x} \, dx = \ln|x| + C$ 6. $\int \sin x \, dx = -\cos x + C$ 7. $\int \cos x \, dx = \sin x + C$ 8. $\int \tan x \, dx = -\ln|\cos x| + C = \ln|\sec x| + C$ 9. $\int \sec x \, dx = \ln|\sec x + \tan x| + C$ 10. $\int \sec^2 x \, dx = \tan x + C$ 11. $\int \sec x \tan x \, dx = \sec x + C$

Substitution Strategy

- Look for composite functions $f(g(x))$ - Check if $g'(x)$ appears (up to a constant multiple) - Let $u = g(x)$, then $du = g'(x)dx$
