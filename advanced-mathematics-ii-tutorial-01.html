Tutorial 1: Complex Numbers

Question 1 [4 marks]

Explain why the set of complex numbers is the universal set for numbers. Draw a Venn diagram to include it, the set of real numbers, the set of irrational numbers, and the set of whole numbers.

Need a hint?

A complex number is in the form a + bi, where a and b are real numbers, and i = ‚àö(-1). Any real number can be written as a + 0i. Therefore, all real numbers are also complex numbers with imaginary part zero.

Show solution

A complex number is in the form $z = a + bi$, where $a, b \in \mathbb{R}$ and $i = \sqrt{-1}$.

Any real number $a$ can be written as $a + 0i$, meaning all real numbers are also complex numbers.

Therefore: - Complex numbers ($\mathbb{C}$) is the universal set - Real numbers ($\mathbb{R}$) ‚äÇ Complex numbers - Rational numbers ($\mathbb{Q}$) ‚äÇ Real numbers - Irrational numbers ‚äÇ Real numbers - Integers ($\mathbb{Z}$) ‚äÇ Rational numbers - Whole numbers (0, 1, 2, 3, ...) ‚äÇ Integers

Venn diagram (from outermost to innermost):

``` Complex (C) ‚îî‚îÄ‚îÄ Real (R) ‚îú‚îÄ‚îÄ Rational (Q) ‚îÇ ‚îî‚îÄ‚îÄ Integer (Z) ‚îÇ ‚îî‚îÄ‚îÄ Whole (0,1,2,3,...) ‚îî‚îÄ‚îÄ Irrational ```

‚ö†Ô∏è Nuance: The imaginary part being zero doesn't make a number "less complex" - it's still a valid complex number.

Question 2 [4 marks]

Solve: (a) $(2x - 1)^2 + 9 = 0$ (b) $x^2 - 2x + 5 = 0$

Need a hint?

For (a): Expand and solve the quadratic, or take square roots. For (b): Use the quadratic formula.

Show solution (a) $(2x - 1)^2 + 9 = 0$

$(2x - 1)^2 = -9$

$2x - 1 = \pm\sqrt{-9} = \pm 3i$

$2x = 1 \pm 3i$

$x = \frac{1 \pm 3i}{2} = \frac{1}{2} \pm \frac{3}{2}i$

(b) $x^2 - 2x + 5 = 0$

Using quadratic formula:

$$x = \frac{-(-2) \pm \sqrt{(-2)^2 - 4(1)(5)}}{2(1)} = \frac{2 \pm \sqrt{4-20}}{2} = \frac{2 \pm \sqrt{-16}}{2} = \frac{2 \pm 4i}{2} = 1 \pm 2i$$

‚ö†Ô∏è Nuance: When discriminant is negative, we get complex conjugate pairs. If $a + bi$ is a solution, $a - bi$ is also a solution.

Question 3 [6 marks]

Find the real and imaginary parts of: (a) $(1 + 2i)(3 - i)$ (b) $\frac{5 - i}{1 - i}$ (c) $i^{-5}$

Need a hint?

For (a): Multiply using FOIL. For (b): Multiply by complex conjugate. For (c): Use $i^4 = 1$.

Show solution (a) $(1 + 2i)(3 - i) = 3 - i + 6i - 2i^2 = 3 + 5i + 2 = 5 + 5i$

Real = 5, Imaginary = 5

(b) $\frac{5 - i}{1 - i} = \frac{(5-i)(1+i)}{(1-i)(1+i)} = \frac{5+5i-i-i^2}{2} = \frac{5+4i+1}{2} = 3 + 2i$

Real = 3, Imaginary = 2

(c) $i^{-5} = \frac{1}{i^5} = \frac{1}{i^4 \cdot i} = \frac{1}{i} = -i$

Real = 0, Imaginary = -1

‚ö†Ô∏è Nuance: For division, always multiply by the complex conjugate. Remember: $i^2 = -1$, $i^3 = -i$, $i^4 = 1$.

Question 4 [4 marks]

Evaluate $i^{4n}$ for $n \in \mathbb{Z}^+$. Hence explain step by step to evaluate $i^n$ for any $n \in \mathbb{Z}^+$.

Need a hint?

Find the pattern: i¬π=i, i¬≤=-1, i¬≥=-i, i‚Å¥=1, then repeat. Use remainder when n divided by 4.

Show solution Powers of i:

- $i^1 = i$ - $i^2 = -1$ - $i^3 = -i$ - $i^4 = 1$ - Cycle repeats every 4

For $i^{4n}$: Since $i^4 = 1$, then $i^{4n} = (i^4)^n = 1^n = 1$ Step-by-step for any $i^n$:

1. Find remainder when n divided by 4: $r = n \mod 4$ 2. Use: - r = 0: $i^n = 1$ - r = 1: $i^n = i$ - r = 2: $i^n = -1$ - r = 3: $i^n = -i$

Example: $i^{25}$ ‚Üí 25 √∑ 4 = 6 remainder 1 ‚Üí $i^{25} = i$

‚ö†Ô∏è Nuance: This cyclic property is essential. Works for negative exponents too.

Question 5 [3 marks]

If $x + iy = (-1 + \sqrt{2}i)(-1 - \sqrt{2}i)$, find $x$ and $y$.

Need a hint?

Multiply complex conjugates: $(a+bi)(a-bi) = a^2 + b^2$.

Show solution

$x + iy = (-1 + \sqrt{2}i)(-1 - \sqrt{2}i) = (-1)^2 + (\sqrt{2})^2 = 1 + 2 = 3$

So $x + iy = 3 + 0i$

Therefore: $x = 3$, $y = 0$

‚ö†Ô∏è Nuance: Product of complex conjugates is always real.

Question 6 [4 marks]

Find the square roots of $z = 2 + 2\sqrt{3}i$.

Need a hint?

Let $w = a + bi$, then $w^2 = z$. So $(a+bi)^2 = (a^2-b^2) + 2abi = 2 + 2\sqrt{3}i$. Equate real and imaginary parts.

Show solution

Let $w = a + bi$ with $w^2 = 2 + 2\sqrt{3}i$

$(a+bi)^2 = (a^2-b^2) + 2abi = 2 + 2\sqrt{3}i$

Equating: - $a^2 - b^2 = 2$ ...(1) - $2ab = 2\sqrt{3}$ ‚Üí $b = \sqrt{3}/a$ ...(2)

Substitute (2) into (1): $a^2 - \frac{3}{a^2} = 2$

$a^4 - 2a^2 - 3 = 0$

$(a^2 - 3)(a^2 + 1) = 0$

$a^2 = 3$ ‚Üí $a = \pm\sqrt{3}$

If $a = \sqrt{3}$: $b = 1$

If $a = -\sqrt{3}$: $b = -1$

Square roots: $\sqrt{3} + i$ and $-\sqrt{3} - i$

‚ö†Ô∏è Nuance: The two square roots are negations of each other.

Question 7 [8 marks]

For each: find modulus & argument (radians), write polar & exponential: (a) $z = 3 - 3\sqrt{3}i$ (b) $z = i\sqrt{3}$ (c) $z = -2\sqrt{3} + 2i$

Need a hint?

$r = \sqrt{x^2+y^2}$, $\theta = \tan^{-1}(y/x)$ adjusted for quadrant.

Show solution (a) $z = 3 - 3\sqrt{3}i$: $r = 6$, $\theta = -\pi/3$

Polar: $6(\cos(-\pi/3) + i\sin(-\pi/3))$, Exponential: $6e^{-i\pi/3}$

(b) $z = i\sqrt{3}$: $r = \sqrt{3}$, $\theta = \pi/2$

Polar: $\sqrt{3}(\cos\pi/2 + i\sin\pi/2)$, Exponential: $\sqrt{3}e^{i\pi/2}$

(c) $z = -2\sqrt{3} + 2i$: $r = 4$, $\theta = 5\pi/6$

Polar: $4(\cos(5\pi/6) + i\sin(5\pi/6))$, Exponential: $4e^{i5\pi/6}$

‚ö†Ô∏è Nuance: Principal argument is in $(-\pi, \pi]$. Check quadrant for correct angle.

Question 8 [2 marks]

If $\text{Im}(z) < 0$ and $\text{Re}(z) = 0$, what is $\arg(z)$?

Need a hint?

Point on negative imaginary axis.

Show solution

$\text{Re}(z) = 0$, $\text{Im}(z) < 0$ ‚Üí negative imaginary axis

$\arg(z) = -\frac{\pi}{2}$ (principal value)

‚ö†Ô∏è Nuance: Points below origin have negative argument.

Question 11 [4 marks]

Find: (a) $\left| \frac{(-2-2i)(1-i\sqrt{3})}{(\sqrt{3}-i)} \right|$ (b) $\arg\left( \frac{3-3i}{(-2+2i)(-1-i)} \right)$

Need a hint?

Use: $|z_1/z_2| = |z_1|/|z_2|$, $\arg(z_1/z_2) = \arg(z_1) - \arg(z_2)$

Show solution (a) $|z| = \frac{|(-2-2i)||(1-i\sqrt{3})|}{|\sqrt{3}-i|} = \frac{2\sqrt{2} \cdot 2}{2} = 2\sqrt{2}$ (b) $\arg(z) = \arg(3-3i) - \arg(-2+2i) - \arg(-1-i)$

Question 12 [4 marks]

Find real and imaginary parts: (a) $3e^{-i\pi/3}$ (b) $2e^{-i3\pi/4}$ (c) $4e^{i5\pi/6}$ (d) $3e^{-i\pi/2}$

Need a hint?

Use $e^{i\theta} = \cos\theta + i\sin\theta$.

Show solution (a) $3e^{-i\pi/3} = 3(\cos(-\pi/3) + i\sin(-\pi/3)) = \frac{3}{2} - \frac{3\sqrt{3}}{2}i$ (b) $2e^{-i3\pi/4} = 2(\cos(-3\pi/4) + i\sin(-3\pi/4)) = -\sqrt{2} + \sqrt{2}i$ (c) $4e^{i5\pi/6} = 4(\cos(5\pi/6) + i\sin(5\pi/6)) = -2\sqrt{3} + 2i$ (d) $3e^{-i\pi/2} = 3(\cos(-\pi/2) + i\sin(-\pi/2)) = 0 - 3i$

‚ö†Ô∏è Nuance: Remember $\sin(-\theta) = -\sin\theta$.

Question 13 [6 marks]

$z_1 = 2e^{-5\pi i/6}$, $z_2 = 4e^{-3\pi i/4}$. Find $|z_1 + z_2|$. Hence show:

$$\cos\frac{11\pi}{12} = -\frac{\sqrt{2} + \sqrt{6}}{4}$$
Need a hint?

Convert to rectangular form, add, find modulus. Use trig sum formula.

Show solution

$z_1 = 2(\cos(-5\pi/6) + i\sin(-5\pi/6)) = -\sqrt{3} - i$

$z_2 = 4(\cos(-3\pi/4) + i\sin(-3\pi/4)) = -2\sqrt{2} - 2\sqrt{2}i$

$z_1 + z_2 = -(\sqrt{3} + 2\sqrt{2}) - (1 + 2\sqrt{2})i$

$|z_1 + z_2| = \sqrt{(\sqrt{3} + 2\sqrt{2})^2 + (1 + 2\sqrt{2})^2}$

Proof:

$\cos(11\pi/12) = \cos(\pi/4 + 2\pi/3) = \cos\pi/4 \cos 2\pi/3 - \sin\pi/4 \sin 2\pi/3$ $= \frac{\sqrt{2}}{2}(-\frac{1}{2}) - \frac{\sqrt{2}}{2}(\frac{\sqrt{3}}{2}) = -\frac{\sqrt{2} + \sqrt{6}}{4}$

‚ö†Ô∏è Nuance: This is a common identity: $\cos(11\pi/12) = -\frac{\sqrt{2} + \sqrt{6}}{4}$.

Common Patterns

Forms:

- Rectangular: $z = x + iy$ - Polar: $z = r(\cos\theta + i\sin\theta)$ - Exponential: $z = re^{i\theta}$

Operations:

- Multiplication: $r_1 r_2 e^{i(\theta_1 + \theta_2)}$ - Division: $\frac{r_1}{r_2} e^{i(\theta_1 - \theta_2)}$

Euler's Formula: $e^{i\theta} = \cos\theta + i\sin\theta$
