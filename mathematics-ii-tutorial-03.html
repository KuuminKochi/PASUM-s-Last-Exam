Tutorial 3: Trigonometric Integrals

Overview

This tutorial covers integration of trigonometric functions using identities, power reduction formulas, and product-to-sum transformations.

Topics: Trigonometric Integrals, Power Reduction Difficulty: Medium Estimated Time: 70 minutes

---

Question 1 [20 marks]

Evaluate the following integrals:

Part (a) [5 marks]

$$\int \tan^7 x \, dx$$
Need a hint?

Use identity $\tan^2 x = \sec^2 x - 1$ repeatedly

Show solution

$$\int \tan^7 x \, dx = \int \tan^5 x (\sec^2 x - 1) dx$$

$= \int \tan^5 x \sec^2 x \, dx - \int \tan^5 x \, dx$

Continue reducing until you get $\int \tan x \, dx = -\ln|\cos x|$

Answer: $\boxed{\frac{\tan^6 x}{6} - \frac{\tan^4 x}{4} + \frac{\tan^2 x}{2} - \ln|\cos x| + C}$

Part (b) [5 marks]

$$\int \cos^5 x \, dx$$
Show solution

$= \int \cos^4 x \cdot \cos x \, dx = \int (1 - \sin^2 x)^2 \cos x \, dx$

Let $u = \sin x$, $du = \cos x \, dx$

$= \int (1 - u^2)^2 du = \int (1 - 2u^2 + u^4) du$

$= \boxed{\sin x - \frac{2}{3}\sin^3 x + \frac{1}{5}\sin^5 x + C}$

Part (c) [5 marks]

$$\int \sin^3 2x \cos^4 2x \, dx$$
Show solution

$= \int \sin^2 2x \cos^4 2x \cdot \sin 2x \, dx$

$= \int (1 - \cos^2 2x)\cos^4 2x \cdot \sin 2x \, dx$

Let $u = \cos 2x$, $du = -2\sin 2x \, dx$

$= -\frac{1}{2}\int (1 - u^2)u^4 du = -\frac{1}{2}\int (u^4 - u^6) du$

$= \boxed{-\frac{\cos^5 2x}{10} + \frac{\cos^7 2x}{14} + C}$

Part (d) [5 marks]

$$\int \tan^5 x \sec^2 x \, dx$$
Show solution

Let $u = \tan x$, $du = \sec^2 x \, dx$

$= \int u^5 du = \boxed{\frac{\tan^6 x}{6} + C}$

Question 2 [10 marks]

Show that $\int \cos^4 x \sin^2 x \, dx = \frac{1}{16}\left(x - \frac{\sin 4x}{4} - \frac{\sin^3 2x}{3}\right) + C$

Show solution

Use power reduction formulas:

$\cos^2 x = \frac{1 + \cos 2x}{2}$, $\sin^2 x = \frac{1 - \cos 2x}{2}$

$\cos^4 x \sin^2 x = \cos^2 x (\cos^2 x \sin^2 x) = \cos^2 x \cdot \frac{\sin^2 2x}{4}$

$= \frac{1 + \cos 2x}{2} \cdot \frac{1 - \cos 4x}{8}$

Expand and integrate term by term...

Verified: RHS differentiates to integrand

Question 3 [20 marks]

Find:

Part (a) [5 marks]

$$\int \sin^3 2x \, dx$$
Show solution

$= \int \sin^2 2x \cdot \sin 2x \, dx = \int (1 - \cos^2 2x)\sin 2x \, dx$

Let $u = \cos 2x$, $du = -2\sin 2x \, dx$

$= -\frac{1}{2}\int (1 - u^2) du = -\frac{u}{2} + \frac{u^3}{6} + C$

$= \boxed{-\frac{\cos 2x}{2} + \frac{\cos^3 2x}{6} + C}$

Part (b) [5 marks]

$$\int \cos^5 ax \, dx$$
Show solution

Similar to Q1(b): $= \int (1 - \sin^2 ax)^2 \cos ax \, dx$

$= \boxed{\frac{\sin ax}{a} - \frac{2\sin^3 ax}{3a} + \frac{\sin^5 ax}{5a} + C}$

Part (c) [5 marks]

$$\int \cos^4 2x \, dx$$
Show solution

Use power reduction twice:

$\cos^4 2x = \left(\frac{1 + \cos 4x}{2}\right)^2 = \frac{1 + 2\cos 4x + \cos^2 4x}{4}$

$= \frac{1}{4}\left(1 + 2\cos 4x + \frac{1 + \cos 8x}{2}\right)$

Integrate: $= \boxed{\frac{3x}{8} + \frac{\sin 4x}{8} + \frac{\sin 8x}{64} + C}$

Part (d) [5 marks]

$$\int \tan^5 3x \, dx$$
Show solution

Similar to Q1(a), use $\tan^2 3x = \sec^2 3x - 1$

$= \boxed{\frac{\tan^4 3x}{12} - \frac{\tan^2 3x}{6} - \frac{\ln|\cos 3x|}{3} + C}$

Question 4 [20 marks]

Evaluate using appropriate techniques:

Part (a) [5 marks]

$$\int \sin^2 x \cos^3 x \, dx$$
Show solution

Odd power of cosine: save one cosine, convert rest to sine

$= \int \sin^2 x \cos^2 x \cdot \cos x \, dx = \int \sin^2 x (1 - \sin^2 x) \cos x \, dx$

Let $u = \sin x$: $= \int u^2(1-u^2) du = \frac{u^3}{3} - \frac{u^5}{5} + C$

$= \boxed{\frac{\sin^3 x}{3} - \frac{\sin^5 x}{5} + C}$

Part (b) [5 marks]

$$\int \sin^3 x \cos^3 x \, dx$$
Show solution

Both odd: convert one to the other

$= \int \sin^3 x \cos^2 x \cdot \cos x \, dx = \int \sin^3 x (1 - \sin^2 x) \cos x \, dx$

$= \boxed{\frac{\sin^4 x}{4} - \frac{\sin^6 x}{6} + C}$

Alternative: Could use $\sin x \cos x = \frac{\sin 2x}{2}$

Part (c) [5 marks]

$$\int \sin^5 x \cos^3 x \, dx$$
Show solution

Odd power of sine: convert to cosine

$= \int \sin^4 x \cos^3 x \cdot \sin x \, dx = \int (1-\cos^2 x)^2 \cos^3 x \sin x \, dx$

$= \boxed{-\frac{\cos^4 x}{4} + \frac{\cos^6 x}{3} - \frac{\cos^8 x}{8} + C}$

Part (d) [5 marks]

$$\int \cos 2x \sin^2 2x \, dx$$
Show solution

Let $u = \sin 2x$, $du = 2\cos 2x \, dx$

$= \frac{1}{2}\int u^2 du = \boxed{\frac{\sin^3 2x}{6} + C}$

Question 5 [30 marks]

Find the following integrals using product-to-sum formulas:

Part (a) [10 marks]

$$\int \sin 2x \cos 4x \, dx$$
Need a hint?

Use product-to-sum: $\sin A \cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]$

Show solution

$\sin 2x \cos 4x = \frac{1}{2}[\sin 6x + \sin(-2x)] = \frac{1}{2}[\sin 6x - \sin 2x]$

$= \frac{1}{2}\left(-\frac{\cos 6x}{6} + \frac{\cos 2x}{2}\right) + C$

$= \boxed{\frac{\cos 2x}{4} - \frac{\cos 6x}{12} + C}$

Part (b) [10 marks]

$$\int \cos 2x \cos 4x \, dx$$
Show solution

$\cos A \cos B = \frac{1}{2}[\cos(A+B) + \cos(A-B)]$

$= \frac{1}{2}[\cos 6x + \cos(-2x)] = \frac{1}{2}[\cos 6x + \cos 2x]$

$= \boxed{\frac{\sin 6x}{12} + \frac{\sin 2x}{4} + C}$

Part (c) [10 marks]

$$\int \sin 2x \sin 4x \, dx$$
Show solution

$\sin A \sin B = \frac{1}{2}[\cos(A-B) - \cos(A+B)]$

$= \frac{1}{2}[\cos(-2x) - \cos 6x] = \frac{1}{2}[\cos 2x - \cos 6x]$

$= \boxed{\frac{\sin 2x}{4} - \frac{\sin 6x}{12} + C}$

‚ö†Ô∏è Nuance: Product-to-sum is essential for $\int \sin mx \cos nx \, dx$ etc.

Common Patterns Identified

Pattern 1: Odd Power Strategy ‚Äî High frequency
Strategy: Save one factor, convert rest to other function using $\sin^2 + \cos^2 = 1$

Pattern 2: Even Power Strategy ‚Äî High frequency
Strategy: Use power reduction formulas: $\cos^2 x = \frac{1+\cos 2x}{2}$

Pattern 3: Product-to-Sum ‚Äî Medium frequency (Q5)
Strategy: Memorize the three product-to-sum formulas

Pattern 4: $\tan^n x$ Integrals ‚Äî Medium frequency
Strategy: Use $\tan^2 x = \sec^2 x - 1$ to reduce power
