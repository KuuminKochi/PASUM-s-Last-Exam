Question 1 [60 marks]

Find the indefinite integrals for the following functions.

Part (a) [5 marks]

$$\\int (3e^{4x} - 5e^{-6x}) \\, dx$$

üí° Concepts: Exponential integration, linearity | Key: $\\int e^{ax} \\, dx = \\frac{1}{a}e^{ax} + C$

Need a hint?

Hint: Split the integral using linearity: $\\int (f(x) - g(x)) \\, dx = \\int f(x) \\, dx - \\int g(x) \\, dx$.

Remember that $\\int e^{ax} \\, dx = \\frac{1}{a}e^{ax} + C$. Watch out for the negative coefficient!

Show solution

Method: Direct integration using exponential rule

$$\\begin{aligned} \\int (3e^{4x} - 5e^{-6x}) \\, dx &= \\frac{3e^{4x}}{4} - \\frac{5e^{-6x}}{-6} + C \\\\ &= \\boxed{\\frac{3}{4}e^{4x} + \\frac{5}{6}e^{-6x} + C} \\end{aligned}$$
‚ö†Ô∏è Critical Nuance: Watch the sign when integrating $e^{-6x}$. The negative exponent becomes positive in the denominator, but the negative sign in front of 5 makes it positive overall. Students often miss this and get $-\\frac{5}{6}$ instead of $+\\frac{5}{6}$.

Part (f) [5 marks]

$$\\int \\frac{\\sin^2 x + \\cos^2 x}{3x} \\, dx$$

üí° Concepts: Trigonometric identity, logarithmic integration

Need a hint?

Hint: Before integrating, look at the numerator. Is there a trigonometric identity that simplifies $\\sin^2 x + \\cos^2 x$?

Show solution

Method: Trigonometric identity + logarithmic integration

$$\\begin{aligned} \\int \\frac{\\sin^2 x + \\cos^2 x}{3x} \\, dx &= \\int \\frac{1}{3x} \\, dx \\quad \\text{(since } \\sin^2 x + \\cos^2 x = 1\\text{)} \\\\ &= \\frac{1}{3} \\int \\frac{1}{x} \\, dx \\\\ &= \\boxed{\\frac{1}{3}\\ln|x| + C} \\end{aligned}$$
‚ö†Ô∏è Critical Nuance: Many students miss the identity $\\sin^2 x + \\cos^2 x = 1$. Without this, the problem becomes much harder. Always look for simplifications before integrating!

Question 2 [40 marks]

Use appropriate substitution to find:

Part (a) [4 marks]

$$\\int 3x^2(x^3 - 2)^3 \\, dx$$

üí° Concepts: u-substitution

Need a hint?

Hint: Try $u = x^3 - 2$. Calculate $du$ and see if $3x^2 \\, dx$ appears in the integral.

Show solution

Method: u-substitution

$$\\begin{aligned} &\\text{Let } u = x^3 - 2, \\quad du = 3x^2 \\, dx \\\\ &\\int 3x^2(x^3 - 2)^3 \\, dx = \\int u^3 \\, du = \\frac{u^4}{4} + C = \\boxed{\\frac{(x^3 - 2)^4}{4} + C} \\end{aligned}$$
‚ö†Ô∏è Critical Nuance: Notice that $3x^2$ is the derivative of $x^3 - 2$. This makes substitution straightforward. Look for this pattern: $f(g(x)) \\cdot g'(x)$
